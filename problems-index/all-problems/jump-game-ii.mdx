      <p>You are given a <strong>0-indexed</strong> array of integers <code>nums</code> of length <code>n</code>. You are initially positioned at&nbsp;index 0.</p>  <p>Each element <code>nums[i]</code> represents the maximum length of a forward jump from index <code>i</code>. In other words, if you are at index <code>i</code>, you can jump to any index <code>(i + j)</code>&nbsp;where:</p>  <ul>  <li><code>0 &lt;= j &lt;= nums[i]</code> and</li>  <li><code>i + j &lt; n</code></li> </ul>  <p>Return <em>the minimum number of jumps to reach index </em><code>n - 1</code>. The test cases are generated such that you can reach index&nbsp;<code>n - 1</code>.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> nums = [2,3,1,1,4] <br/> <strong>Output:</strong> 2 <br/> <strong>Explanation:</strong> The minimum number of jumps to reach the last index is 2. # @param {Integer[]} nums
# @return {Integer}
def jump(nums)
  jumps = 0
  left_p = right_p = 0
  n = nums.size
  return 0 if n < 2
  
  # if we reach n-1, no more processing, its done
  while right_p < n-1
    farthest_in_current_jump  = 0
    (left_p..right_p).each { |idx| farthest_in_current_jump = [farthest_in_current_jump, idx + nums[idx] ].max }
    
    left_p = right_p + 1
    right_p = farthest_in_current_jump
    jumps+=1 # if right_p reaches n-1, jump value will be 0
  end
  jumps
end
