      <p>A <strong>ramp</strong> in an integer array <code>nums</code> is a pair <code>(i, j)</code> for which <code>i &lt; j</code> and <code>nums[i] &lt;= nums[j]</code>. The <strong>width</strong> of such a ramp is <code>j - i</code>.</p>  <p>Given an integer array <code>nums</code>, return <em>the maximum width of a <strong>ramp</strong> in </em><code>nums</code>. If there is no <strong>ramp</strong> in <code>nums</code>, return <code>0</code>.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> nums = [6,0,8,2,1,5] <br/> <strong>Output:</strong> 4 <br/> <strong>Explanation:</strong> The maximum width ramp is achieved at (i, j) = (1, 5): nums[1] = 0 and nums[5] = 5.   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> nums = [9,8,1,0,1,9,4,0,4,1] <br/> <strong>Output:</strong> 7 <br/> <strong>Explanation:</strong> The maximum width ramp is achieved at (i, j) = (2, 9): nums[2] = 1 and nums[9] = 1.    <p><strong>Constraints:</strong></p>  <ul>  <li><code>2 &lt;= nums.length &lt;= 5 * 10<sup>4</sup></code></li>  <li><code>0 &lt;= nums[i] &lt;= 5 * 10<sup>4</sup></code></li> </ul>

  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      Just sort idxs based on num at that idx, then get max diff possible.
</Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Maximum Width Ramp" lines
# @param {Integer[]} nums
# @return {Integer}
def max_width_ramp(nums)
  sorted_idxs = nums.map.with_index.sort_by {|x| x[0]}.map(&:last)

  sorted_idxs.reduce([Float::INFINITY, 0]) do |(min_idx, result), i|
    [[min_idx, i].min, [result, i-min_idx].max]
  end[1]
end
```
