The probability of each element being returned is <strong>linearly related</strong> to the number of the same values the multiset contains.</li> </ul>  <p>You must implement the functions of the class such that each function works on <strong>average</strong> <code>O(1)</code> time complexity.</p>  <p><strong>Note:</strong> The test cases are generated such that <code>getRandom</code> will only be called if there is <strong>at least one</strong> item in the <code>RandomizedCollection</code>.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input</strong> [&quot;RandomizedCollection&quot;, &quot;insert&quot;, &quot;insert&quot;, &quot;insert&quot;, &quot;getRandom&quot;, &quot;remove&quot;, &quot;getRandom&quot;] [[], [1], [1], [2], [], [1], []] <strong>Output</strong> [null, true, false, true, 2, true, 1]  <strong>Explanation</strong> RandomizedCollection randomizedCollection = new RandomizedCollection(); randomizedCollection.insert(1);   // return true since the collection does not contain 1.                                   // Inserts 1 into the collection. randomizedCollection.insert(1);   // return false since the collection contains 1.                                   // Inserts another 1 into the collection.   <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      In order to have O(1) removal from a list, we can use hashmap, that is the trick here. 

Once this is done, we can simply store all items in the list, and pick a random index, while inserting, we insert at end. While popping, we pop the swap the last element, and pop.

Similar to https://leetcode.com/problems/insert-delete-getrandom-o1/. In this problem, we can have multiple idxs, so store those in the hash instead of just 1 idx.
    </Card>

    <CodeGroup dropdown>
```ruby title="Insert Delete GetRandom O(1) - Duplicates allowed" lines
class RandomizedCollection
    def initialize()
      @hash = {}
      @arr = []
    end


=begin
    :type val: Integer
    :rtype: Boolean
=end
    def insert(val)
      (@hash[val] ||= Set.new).add(@arr.size)
      @arr << val
      @hash[val].size == 1
    end


=begin
    :type val: Integer
    :rtype: Boolean
=end
    def remove(val)
      return false if (@hash[val] ||= Set.new).empty?
      
      # idx of element being deleted
      index, last = @hash[val].first, @arr.last
      
      # swap, then pop, imp for same element being added/removed
      # instead of pop and swap
      @arr[index] = last; @arr.pop
      @hash[val].delete(index)
      
      # update last elements idx from last to idx of deleted element
      @hash[last].add(index)
      @hash[last].delete(@arr.size)
      true
    end


=begin
    :rtype: Integer
=end
    def get_random()
      @arr[rand(@arr.size)]
    end


end
```
