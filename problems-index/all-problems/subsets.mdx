Classic way of enumerating submasks or a neat-tricky version is as mentioned here

This is to generate all possible sub-masks for a given bitmask.

```ruby title="Subsets" lines
# @param {Integer[]} nums
# @return {Integer[][]}
def subsets(nums)
  @results = []
  backtrack = ->(nums, current_subset, index) do
    if index == nums.size
      @results << current_subset.dup
      return
    end
    
    current_subset << nums[index]
    backtrack.call(nums, current_subset.dup, index+1)
    current_subset.pop
    backtrack.call(nums, current_subset.dup, index+1)    
  end

  backtrack.call(nums, [], 0)
  @results
end
```
