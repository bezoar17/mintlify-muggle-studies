      <p>Given the <code>root</code> of a binary tree, return <em><strong>the vertical order traversal</strong> of its nodes&#39; values</em>. (i.e., from top to bottom, column by column).</p>  <p>If two nodes are in the same row and column, the order should be from <strong>left to right</strong>.</p>   <p><strong class="example">Example 1:</strong></p> <img alt="" src="https://assets.leetcode.com/uploads/2024/09/23/image1.png" style="width: 400px; height: 273px;" />  <strong>Input:</strong> root = [3,9,20,null,null,15,7] <br/> <strong>Output:</strong> [[9],[3,15],[20],[7]]   <p><strong class="example">Example 2:</strong></p> <img alt="" src="https://assets.leetcode.com/uploads/2024/09/23/image3.png" style="width: 450px; height: 285px;" />  <strong>Input:</strong> root = [3,9,8,4,0,1,7] <br/> <strong>Output:</strong> [[4],[9],[3,0,1],[8],[7]]   <p><strong class="example">Example 3:</strong></p> <img alt="" src="https://assets.leetcode.com/uploads/2024/09/23/image2.png" style="width: 350px; height: 342px;" />  <strong>Input:</strong> root = [1,2,3,4,10,9,11,null,5,null,null,null,null,null,null,null,6] <br/> <strong>Output:</strong> [[4],[2,5],[1,10,9,6],[3],[11]]    <p><strong>Constraints:</strong></p>  <ul>  <li>The number of nodes in the tree is in the range <code>[0, 100]</code>.</li>  <li><code>-100 &lt;= Node.val &lt;= 100</code></li> </ul>

  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      BFS and append values in the column hash, for each col. Dynamically assign col values, col-1 to left node, and col+1 to right one.
    </Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Binary Tree Vertical Order Traversal" lines
# Definition for a binary tree node.
# class TreeNode
#     attr_accessor :val, :left, :right
#     def initialize(val = 0, left = nil, right = nil)
#         @val = val
#         @left = left
#         @right = right
#     end
# end
# @param {TreeNode} root
# @return {Integer[][]}
def vertical_order(root)
  col_h = Hash.new {|h,k| h[k]=[]}
  return [] if root.nil?
  # bfs
  queue = []
  queue << [root, 0]
  until queue.empty?
    node, col = queue.shift
    col_h[col] << node.val
    queue << [node.left, col-1] unless node.left.nil?
    queue << [node.right, col+1] unless node.right.nil?
  end

  col_h.keys.sort.map {|k| col_h[k] }
end
```
