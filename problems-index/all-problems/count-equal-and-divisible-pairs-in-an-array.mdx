      Given a <strong>0-indexed</strong> integer array <code>nums</code> of length <code>n</code> and an integer <code>k</code>, return <em>the <strong>number of pairs</strong></em> <code>(i, j)</code> <em>where</em> <code>0 &lt;= i &lt; j &lt; n</code>, <em>such that</em> <code>nums[i] == nums[j]</code> <em>and</em> <code>(i * j)</code> <em>is divisible by</em> <code>k</code>.  <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> nums = [3,1,2,2,2,1,3], k = 2 <br/> <strong>Output:</strong> 4 <br/> <strong>Explanation:</strong> There are 4 pairs that meet all the requirements: - nums[0] == nums[6], and 0 * 6 == 0, which is divisible by 2. - nums[2] == nums[3], and 2 * 3 == 6, which is divisible by 2. - nums[2] == nums[4], and 2 * 4 == 8, which is divisible by 2. - nums[3] == nums[4], and 3 * 4 == 12, which is divisible by 2.   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> nums = [1,2,3,4], k = 1 <br/> <strong>Output:</strong> 0 <br/> <strong>Explanation:</strong> Since no value in nums is repeated, there are no pairs (i,j) that meet all the requirements.    <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= nums.length &lt;= 100</code></li>  <li><code>1 &lt;= nums[i], k &lt;= 100</code></li> </ul>

```ruby title="Count Equal and Divisible Pairs in an Array" lines
# @param {Integer[]} nums
# @param {Integer} k
# @return {Integer}
def count_pairs(nums, k)
  n = nums.size
  count = 0 
  (0...n).each do |i|
    (0...i).each do |j|
      count+=1 if nums[i] == nums[j] && ((i*j)%k == 0)
    end
  end
  count
end
```
