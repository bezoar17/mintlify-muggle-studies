    <Card>
      Our array looks like f1,d1,f2,d2,f3,d3,f4, here f can be 0 also. 
If we have to move 2, d1d2 or d2d3 can be moved, consecutive have to be moved, to maximize free time. If k was == n, no. of events, then all are moved and we consume all the free time. 
1 move sums its alternate values, so, if k=1, we can combine any 2 consecutive f values, sub-array sum max of 2 elements.

For n meetings, we have n+1 free time values. For merging k elements, we merge k+1 consecutive meetings. Find sliding window sum of k+1 free time values
</Card>

```ruby title="Reschedule Meetings for Maximum Free Time I" lines
# @param {Integer} event_time
# @param {Integer} k
# @param {Integer[]} start_time
# @param {Integer[]} end_time
# @return {Integer}
Meeting = Struct.new(:start_time, :end_time)
def max_free_time(event_time, k, start_time, end_time)
  meetings = start_time.zip(end_time).map {|a,b|Meeting.new(a,b) }
  n = meetings.size
  free_times = [meetings[0].start_time-0]
  (1...n).each {|i| free_times << meetings[i].start_time-meetings[i-1].end_time }
  free_times << (event_time - meetings[-1].end_time)

  current_sum = free_times.first(k+1).sum
  result = current_sum

  ((k+1)..n).each do |i| # sub-array sum ending at i in free_times array
    current_sum += free_times[i]-free_times[i-k-1]
    result = [current_sum, result].max
  end
  result
end
```
