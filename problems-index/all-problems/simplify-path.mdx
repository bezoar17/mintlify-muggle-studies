  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      Classic stack problem, to get dirs, first use split('/').

Then // can be ignored as they become ''
'.' can anyways be ignored
'..' we can pop from stack

then return / + stack.join('/'), this takes care of root thing.
    </Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Simplify Path" lines
# @param {String} path
# @return {String}
def simplify_path(path)
  stack = []
  dirs = path.split('/')
  
  dirs.each_with_index do |dir, idx|
    # ignore . and ignore multiple // which create '' after splits
    next if dir.eql?('.') || dir.eql?('') 
    if dir.eql?('..') # jump to parent, so remove top of stack
      stack.pop unless stack.empty?
    else
      stack << dir
    end
  end
  
  '/' + stack.join('/')
end
```
