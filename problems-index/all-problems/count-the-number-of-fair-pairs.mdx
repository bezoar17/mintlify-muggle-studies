      <p>Given a <strong>0-indexed</strong> integer array <code>nums</code> of size <code>n</code> and two integers <code>lower</code> and <code>upper</code>, return <em>the number of fair pairs</em>.</p>  <p>A pair <code>(i, j)</code> is <b>fair </b>if:</p>  <ul>  <li><code>0 &lt;= i &lt; j &lt; n</code>, and</li>  <li><code>lower &lt;= nums[i] + nums[j] &lt;= upper</code></li> </ul>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> nums = [0,1,7,4,4,5], lower = 3, upper = 6 <br/> <strong>Output:</strong> 6 <br/> <strong>Explanation:</strong> There are 6 fair pairs: (0,3), (0,4), (0,5), (1,3), (1,4), and (1,5).   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> nums = [1,7,9,2,5], lower = 11, upper = 11 <br/> <strong>Output:</strong> 1 <br/> <strong>Explanation:</strong> There is a single fair pair: (2,3).    <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= nums.length &lt;= 10<sup>5</sup></code></li>  <li><code>nums.length == n</code></li>  <li><code><font face="monospace">-10<sup>9</sup></font>&nbsp;&lt;= nums[i] &lt;= 10<sup>9</sup></code></li>  <li><code><font face="monospace">-10<sup>9</sup>&nbsp;&lt;= lower &lt;= upper &lt;= 10<sup>9</sup></font></code></li> </ul>

# @param {Integer[]} nums
# @param {Integer} lower
# @param {Integer} upper
# @return {Integer}
def count_fair_pairs(nums, lower, upper)
  nums.sort!
  count_pair_with(nums, upper) - count_pair_with(nums, lower - 1)
end

def count_pair_with(nums, sum)
  res = 0
  left = 0
  right = nums.size - 1
  while left < right
    right -= 1 while right > left && nums[left] + nums[right] > sum
    res += right - left
    left += 1
  end
  res
end
