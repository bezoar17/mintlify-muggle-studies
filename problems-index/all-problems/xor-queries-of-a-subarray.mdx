      <p>You are given an array <code>arr</code> of positive integers. You are also given the array <code>queries</code> where <code>queries[i] = [left<sub>i, </sub>right<sub>i</sub>]</code>.</p>  <p>For each query <code>i</code> compute the <strong>XOR</strong> of elements from <code>left<sub>i</sub></code> to <code>right<sub>i</sub></code> (that is, <code>arr[left<sub>i</sub>] XOR arr[left<sub>i</sub> + 1] XOR ... XOR arr[right<sub>i</sub>]</code> ).</p>  <p>Return an array <code>answer</code> where <code>answer[i]</code> is the answer to the <code>i<sup>th</sup></code> query.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> arr = [1,3,4,8], queries = [[0,1],[1,2],[0,3],[3,3]] <br/> <strong>Output:</strong> [2,7,14,8]  <br/> <strong>Explanation:</strong>  The binary representation of the elements in the array are: 1 = 0001  3 = 0011  4 = 0100  8 = 1000  The XOR values for queries are: [0,1] = 1 xor 3 = 2  [1,2] = 3 xor 4 = 7  [0,3] = 1 xor 3 xor 4 xor 8 = 14  [3,3] = 8   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> arr = [4,8,2,10], queries = [[2,3],[1,3],[0,0],[0,3]] <br/> <strong>Output:</strong> [8,0,4,4]    <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= arr.length, queries.length &lt;= 3 * 10<sup>4</sup></code></li>  <li><code>1 &lt;= arr[i] &lt;= 10<sup>9</sup></code></li>  <li><code>queries[i].length == 2</code></li>  <li><code>0 &lt;= left<sub>i</sub> &lt;= right<sub>i</sub> &lt; arr.length</code></li> </ul>

    <Card>
      For each bit in max_bit_length, maintain a prefix sum of elements in arr. This will give us for a bit_pos, the no. of 1s between [l,r]. If thats odd, bit_pos will be set in result, else will be 0.

This can also be done by storing XOR prefixes till i. Instead of storing counts of 1 at each bit. 
[a,b,c,d,e,f]. XOR[b] till b is a^b. XOR[e] till e is a^b^c^d^e. If we need xor from c to e. We just XOR[b] with XOR[e]. More specifically, to get XOR[l+1, r], we do XOR[0..r] ^ XOR[0..l]. All nums till [0..l] will repeat and cancel out, the remaining will be XOR[l+1, r], which is our answer.


Both solutions are present in submissions.
</Card>

```ruby title="XOR Queries of a Subarray" lines
# @param {Integer[]} arr
# @param {Integer[][]} queries
# @return {Integer[]}
def xor_queries(arr, queries)
  max_bit_length = arr.map(&:bit_length).max
  n = arr.size
  xor_prefix = Array.new(n+1, 0) # xor_prefix[i] is xor of elements [0..i-1]
  (1..n).each {|i| xor_prefix[i] = arr[i-1] ^ xor_prefix[i-1] }
  queries.map {|l,r| xor_prefix[r+1]^xor_prefix[l] }
end
```
