  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      Intialize a transformations array to store the no. of transformations for each index
Mark the queries onto transformations array, for each query l,r, mark l as +1, and r+1 as -1.

Suppose only 1 query was there, then go from left to right in transformations array, and accumulate the transformations for each index, if only 1 query was there, only l index would have +1, which will remain till r index, and after r it will become 0.

We are essentially capturing the transformations, instead of applying it for each query, we just store the affected range by mapping the start and end points of the query.
</Card>

```ruby title="Zero Array Transformation I" lines
# @param {Integer[]} nums
# @param {Integer[][]} queries
# @return {Boolean}
def is_zero_array(nums, queries)
  n = nums.size
  transformation_markers = Array.new(n+1, 0)
  queries.each do |left, right|
    transformation_markers[left] += 1
    transformation_markers[right+1] -= 1 if right+1 <= n
  end

  current_deficit = 0

  transformation_markers[0..-2].each_with_index do |deficit, idx|
    current_deficit += deficit
    return false if nums[idx] > current_deficit
  end
  true
end
```
