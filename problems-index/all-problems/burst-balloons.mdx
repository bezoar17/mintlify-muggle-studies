      <p>You are given <code>n</code> balloons, indexed from <code>0</code> to <code>n - 1</code>. Each balloon is painted with a number on it represented by an array <code>nums</code>. You are asked to burst all the balloons.</p>  <p>If you burst the <code>i<sup>th</sup></code> balloon, you will get <code>nums[i - 1] * nums[i] * nums[i + 1]</code> coins. If <code>i - 1</code> or <code>i + 1</code> goes out of bounds of the array, then treat it as if there is a balloon with a <code>1</code> painted on it.</p>  <p>Return <em>the maximum coins you can collect by bursting the balloons wisely</em>.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> nums = [3,1,5,8] <br/> <strong>Output:</strong> 167 <br/> <strong>Explanation:</strong> nums = [3,1,5,8] --&gt; [3,5,8] --&gt; [3,8] --&gt; [8] --&gt; [] coins =  3*1*5    +   3*5*8   +  1*3*8  + 1*8*1 = 167  <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> nums = [1,5] <br/> <strong>Output:</strong> 10    <p><strong>Constraints:</strong></p>  <ul>  <li><code>n == nums.length</code></li>  <li><code>1 &lt;= n &lt;= 300</code></li>  <li><code>0 &lt;= nums[i] &lt;= 100</code></li> </ul>

Popping a balloon first, doesnt give independent subproblems, 
but popping a balloon last does. 

Considering kth index to be popped last, the subproblems will include kth balloon as the imaginary 1s in left and right boundaries while calculating the subproblems.
</Card>

```ruby title="Burst Balloons" lines
# @param {Integer[]} nums
# @return {Integer}
def max_coins(nums)
  n = nums.length
  nums = [1] + nums + [1]
  dp = Array.new(n+2) { Array.new(n+2, 0) }

  (n).downto(1).each do |left|
    (left..n).each do |right|
      (left..right).each do |i|
        coins = nums[left - 1] * nums[i] * nums[right + 1]
        coins += dp[left][i - 1] + dp[i + 1][right]
        dp[left][right] = [dp[left][right], coins].max
      end
    end
  end
  dp[1][n]
end
```
