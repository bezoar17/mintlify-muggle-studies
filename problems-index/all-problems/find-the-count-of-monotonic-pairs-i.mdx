      <p>You are given an array of <strong>positive</strong> integers <code>nums</code> of length <code>n</code>.</p>  <p>We call a pair of <strong>non-negative</strong> integer arrays <code>(arr1, arr2)</code> <strong>monotonic</strong> if:</p>  <ul>  <li>The lengths of both arrays are <code>n</code>.</li>  <li><code>arr1</code> is monotonically <strong>non-decreasing</strong>, in other words, <code>arr1[0] &lt;= arr1[1] &lt;= ... &lt;= arr1[n - 1]</code>.</li>  <li><code>arr2</code> is monotonically <strong>non-increasing</strong>, in other words, <code>arr2[0] &gt;= arr2[1] &gt;= ... &gt;= arr2[n - 1]</code>.</li>  <li><code>arr1[i] + arr2[i] == nums[i]</code> for all <code>0 &lt;= i &lt;= n - 1</code>.</li> </ul>  <p>Return the count of <strong>monotonic</strong> pairs.</p>  <p>Since the answer may be very large, return it <strong>modulo</strong> <code>10<sup>9</sup> + 7</code>.</p>   <p><strong class="example">Example 1:</strong></p>  <div class="example-block"> <p><strong>Input:</strong> <span class="example-io">nums = [2,3,2]</span></p>  <p><br/> <strong>Output:</strong> <span class="example-io">4</span></p>  <p><br/> <strong>Explanation:</strong></p>  <p>The good pairs are:</p>  <ol>  <li><code>([0, 1, 1], [2, 2, 1])</code></li>  <li><code>([0, 1, 2], [2, 2, 0])</code></li>  <li><code>([0, 2, 2], [2, 1, 0])</code></li>  <li><code>([1, 2, 2], [1, 1, 0])</code></li> </ol> </div>  <p><strong class="example">Example 2:</strong></p>  <div class="example-block"> <p><strong>Input:</strong> <span class="example-io">nums = [5,5,5,5]</span></p>  <p><br/> <strong>Output:</strong> <span class="example-io">126</span></p> </div>   <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= n == nums.length &lt;= 2000</code></li>  <li><code>1 &lt;= nums[i] &lt;= 50</code></li> </ul>

  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
        Suppose arr1 is a, arr2 is b
  based on last value in a, an elements range is last_a ..
	and based on last value in b, its range is  ..last_b
  and last value in b is nums[i-1] - last_value_in_a
	
	So we only need to track last val in a and index i.
	
	Simulate the dp as range is small.
    </Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Find the Count of Monotonic Pairs I" lines
# @param {Integer[]} nums
# @return {Integer}
def count_of_pairs(nums)
  n = nums.size; @n = n
  @max_num = 50 # given as limit in test case
  @nums = nums
  @dp = Array.new(n) { Array.new(@max_num+1) }
  @mod = 1_000_000_000 + 7

  recursive(0, 0) % @mod
end

def recursive(i, last_val_in_a)
  return 1 if i>=@n

  return @dp[i][last_val_in_a] unless @dp[i][last_val_in_a].nil?

  last_val_in_b = i==0 ? @max_num+1 : (@nums[i-1] - last_val_in_a)
  a_range = (last_val_in_a..@nums[i]).select {|x| @nums[i]-x <= last_val_in_b }

  @dp[i][last_val_in_a] = a_range.map { |a_val| recursive(i+1, a_val) }.sum % @mod
end
```
