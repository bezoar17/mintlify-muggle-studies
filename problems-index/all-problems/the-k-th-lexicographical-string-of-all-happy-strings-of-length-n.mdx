    <Card>
      For a string of length n, first char has 3 possibilities, rest all have 2. So, we have 3* 2^(n-1) strings in total. We first fix the first char, and offset k accordingly. Now, for each next char, if k is in left half, we take the next char, else we take next to next char, and offset k, for all remainder of the chars.

https://leetcode.com/problems/kth-smallest-instructions/
</Card>

```ruby title="The k-th Lexicographical String of All Happy Strings of Length n" lines
# @param {Integer} n
# @param {Integer} k
# @return {String}
def get_happy_string(n, k)
  # 3 * 2 ^ n-1 strings in total
  return "" if k > 3*(2**(n-1))
  
  chars = ['a','b','c']
  rem = n-1
  rem_p = 2**rem # possible chars for remaining chars
  first_char_idx = (k-1)/rem_p # just to get idx in char, we use k-1
  str = [chars[first_char_idx]]
  k-= (first_char_idx)*rem_p
  
  rem.times do
    if 2*k <= rem_p
      str << (str.last!='a' ? 'a' : 'b') # left half
    else
      str << (str.last!='c' ? 'c' : 'b') # right half
      k-= rem_p/2
    end
    rem_p/=2
  end

  str.join
end
```
