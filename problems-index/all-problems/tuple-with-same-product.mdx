    <Card>
      <p>Given an array <code>nums</code> of <strong>distinct</strong> positive integers, return <em>the number of tuples </em><code>(a, b, c, d)</code><em> such that </em><code>a * b = c * d</code><em> where </em><code>a</code><em>, </em><code>b</code><em>, </em><code>c</code><em>, and </em><code>d</code><em> are elements of </em><code>nums</code><em>, and </em><code>a != b != c != d</code><em>.</em></p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> nums = [2,3,4,6] <br/> <strong>Output:</strong> 8 <br/> <strong>Explanation:</strong> There are 8 valid tuples: (2,6,3,4) , (2,6,4,3) , (6,2,3,4) , (6,2,4,3) (3,4,2,6) , (4,3,2,6) , (3,4,6,2) , (4,3,6,2)   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> nums = [1,2,4,5,10] <br/> <strong>Output:</strong> 16 <br/> <strong>Explanation:</strong> There are 16 valid tuples: (1,10,2,5) , (1,10,5,2) , (10,1,2,5) , (10,1,5,2) (2,5,1,10) , (2,5,10,1) , (5,2,1,10) , (5,2,10,1) (2,10,4,5) , (2,10,5,4) , (10,2,4,5) , (10,2,5,4) (4,5,2,10) , (4,5,10,2) , (5,4,2,10) , (5,4,10,2)    <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= nums.length &lt;= 1000</code></li>  <li><code>1 &lt;= nums[i] &lt;= 10<sup>4</sup></code></li>  <li>All elements in <code>nums</code> are <strong>distinct</strong>.</li> </ul> 
    </Card>

  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      For (a,b,c,d), we have 8 tuples. For each product value if we find P pairs, we can have PC2 such abcd values, and for each we have 8 tuples. So, ***p  (p-1)  4*** is the answer, where p is pair count for each product value.
    </Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Tuple with Same Product" lines
# @param {Integer[]} nums
# @return {Integer}
def tuple_same_product(nums)
  # For each product value if we find P pairs, we have 8 tuples for abcd, and we can have PC2 such values
  # 120, 1; 8,15; 10,12, all are 120 product, pick any 2 pairs, and we have 8 tuples for them
  product_h = Hash.new(0)
  n = nums.size
  (0...n).each { |i| ((i+1)...n).each {|j| product_h[nums[i]*nums[j]]+=1 }}
  product_h.values.sum { |pair_count| pair_count*(pair_count-1)*4 } # PC2 * 8
end
```
