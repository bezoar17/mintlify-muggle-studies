      Given the array of integers <code>nums</code>, you will choose two different indices <code>i</code> and <code>j</code> of that array. <em>Return the maximum value of</em> <code>(nums[i]-1)*(nums[j]-1)</code>.  <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> nums = [3,4,5,2] <br/> <strong>Output:</strong> 12  <br/> <strong>Explanation:</strong> If you choose the indices i=1 and j=2 (indexed from 0), you will get the maximum value, that is, (nums[1]-1)*(nums[2]-1) = (4-1)*(5-1) = 3*4 = 12.    <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> nums = [1,5,4,5] <br/> <strong>Output:</strong> 16 <br/> <strong>Explanation:</strong> Choosing the indices i=1 and j=3 (indexed from 0), you will get the maximum value of (5-1)*(5-1) = 16.   <p><strong class="example">Example 3:</strong></p>   <strong>Input:</strong> nums = [3,7] <br/> <strong>Output:</strong> 12    <p><strong>Constraints:</strong></p>  <ul>  <li><code>2 &lt;= nums.length &lt;= 500</code></li>  <li><code>1 &lt;= nums[i] &lt;= 10^3</code></li> </ul>

```ruby title="Maximum Product of Two Elements in an Array" lines
# @param {Integer[]} nums
# @return {Integer}
def max_product(nums)
  nums.sort.last(2).map{_1-1}.reduce(&:*)
end
```
