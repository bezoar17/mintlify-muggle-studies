      <p>There are <code>n</code> people standing in a queue, and they numbered from <code>0</code> to <code>n - 1</code> in <strong>left to right</strong> order. You are given an array <code>heights</code> of <strong>distinct</strong> integers where <code>heights[i]</code> represents the height of the <code>i<sup>th</sup></code> person.</p>  <p>A person can <strong>see</strong> another person to their right in the queue if everybody in between is <strong>shorter</strong> than both of them. More formally, the <code>i<sup>th</sup></code> person can see the <code>j<sup>th</sup></code> person if <code>i &lt; j</code> and <code>min(heights[i], heights[j]) &gt; max(heights[i+1], heights[i+2], ..., heights[j-1])</code>.</p>  <p>Return <em>an array </em><code>answer</code><em> of length </em><code>n</code><em> where </em><code>answer[i]</code><em> is the <strong>number of people</strong> the </em><code>i<sup>th</sup></code><em> person can <strong>see</strong> to their right in the queue</em>.</p>   <p><strong class="example">Example 1:</strong></p>  <p><img alt="" src="https://assets.leetcode.com/uploads/2021/05/29/queue-plane.jpg" style="width: 600px; height: 247px;" /></p>   <strong>Input:</strong> heights = [10,6,8,5,11,9] <br/> <strong>Output:</strong> [3,1,2,1,1,0] <br/> <strong>Explanation:</strong> Person 0 can see person 1, 2, and 4. Person 1 can see person 2. Person 2 can see person 3 and 4. Person 3 can see person 4. Person 4 can see person 5. Person 5 can see no one since nobody is to the right of them.   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> heights = [5,1,2,3,10] <br/> <strong>Output:</strong> [4,1,1,1,0]    <p><strong>Constraints:</strong></p>  <ul>  <li><code>n == heights.length</code></li>  <li><code>1 &lt;= n &lt;= 10<sup>5</sup></code></li>  <li><code>1 &lt;= heights[i] &lt;= 10<sup>5</sup></code></li>  <li>All the values of <code>heights</code> are <strong>unique</strong>.</li> </ul>

    <Card>
      Classic monotonic stack question, get next greater_eq element, idx cant see beyond that,
answer is number of discarded values while calculating next ge element + element being added.
Whatever you pop is smaller, hence counts as people seen.
    </Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Number of Visible People in a Queue" lines
# @param {Integer[]} heights
# @return {Integer[]}
def can_see_persons_count(heights)
  n = heights.size
  result = Array.new(n, 0)
  stack = []

  discard =->(top, idx) { heights[top] < heights[idx] }

  (n-1).downto(0).each do |idx|
    count = 0
    while !stack.empty? && discard.call(stack.last, idx)
      count+=1
      stack.pop
    end
    count+=1 unless stack.empty?
    result[idx] = count
    stack << idx
  end
  result
end
```
