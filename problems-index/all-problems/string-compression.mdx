      <p>Given an array of characters <code>chars</code>, compress it using the following algorithm:</p>  <p>Begin with an empty string <code>s</code>. For each group of <strong>consecutive repeating characters</strong> in <code>chars</code>:</p>  <ul>  <li>If the group&#39;s length is <code>1</code>, append the character to <code>s</code>.</li>  <li>Otherwise, append the character followed by the group&#39;s length.</li> </ul>  <p>The compressed string <code>s</code> <strong>should not be returned separately</strong>, but instead, be stored <strong>in the input character array <code>chars</code></strong>. Note that group lengths that are <code>10</code> or longer will be split into multiple characters in <code>chars</code>.</p>  <p>After you are done <strong>modifying the input array,</strong> return <em>the new length of the array</em>.</p>  <p>You must write an algorithm that uses only constant extra space.</p>  <p><strong>Note: </strong>The characters in the array beyond the returned length do not matter and should be ignored.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> chars = [&quot;a&quot;,&quot;a&quot;,&quot;b&quot;,&quot;b&quot;,&quot;c&quot;,&quot;c&quot;,&quot;c&quot;] <br/> <strong>Output:</strong> Return 6, and the first 6 characters of the input array should be: [&quot;a&quot;,&quot;2&quot;,&quot;b&quot;,&quot;2&quot;,&quot;c&quot;,&quot;3&quot;] <br/> <strong>Explanation:</strong> The groups are &quot;aa&quot;, &quot;bb&quot;, and &quot;ccc&quot;. This compresses to &quot;a2b2c3&quot;.   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> chars = [&quot;a&quot;] <br/> <strong>Output:</strong> Return 1, and the first character of the input array should be: [&quot;a&quot;] <br/> <strong>Explanation:</strong> The only group is &quot;a&quot;, which remains uncompressed since it&#39;s a single character.   <p><strong class="example">Example 3:</strong></p>   <strong>Input:</strong> chars = [&quot;a&quot;,&quot;b&quot;,&quot;b&quot;,&quot;b&quot;,&quot;b&quot;,&quot;b&quot;,&quot;b&quot;,&quot;b&quot;,&quot;b&quot;,&quot;b&quot;,&quot;b&quot;,&quot;b&quot;,&quot;b&quot;] <br/> <strong>Output:</strong> Return 4, and the first 4 characters of the input array should be: [&quot;a&quot;,&quot;b&quot;,&quot;1&quot;,&quot;2&quot;]. <br/> <strong>Explanation:</strong> The groups are &quot;a&quot; and &quot;bbbbbbbbbbbb&quot;. ```ruby title="String Compression" lines
# @param {Character[]} chars
# @return {Integer}
def compress(chars)
  current_pointer = 0
  n = chars.size
  update_pointer = 0

  while current_pointer < n
    current_char = chars[current_pointer]
    size = 1
    while current_pointer + 1 < n && current_char == chars[current_pointer+1]
      size +=1
      current_pointer += 1
    end

    chars[update_pointer] = current_char
    update_pointer+=1
    
    if size > 1
      size.to_s.each_char { |ch|
        chars[update_pointer] = ch
        update_pointer+=1
      }
    end
    
    current_pointer += 1
  end

  update_pointer
end
