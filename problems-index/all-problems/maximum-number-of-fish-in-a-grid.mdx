0</code>.</li> </ul>  <p>A fisher can start at any <strong>water</strong> cell <code>(r, c)</code> and can do the following operations any number of times:</p>  <ul>  <li>Catch all the fish at cell <code>(r, c)</code>, or</li>  <li>Move to any adjacent <strong>water</strong> cell.</li> </ul>  <p>Return <em>the <strong>maximum</strong> number of fish the fisher can catch if he chooses his starting cell optimally, or </em><code>0</code> if no water cell exists.</p>  <p>An <strong>adjacent</strong> cell of the cell <code>(r, c)</code>, is one of the cells <code>(r, c + 1)</code>, <code>(r, c - 1)</code>, <code>(r + 1, c)</code> or <code>(r - 1, c)</code> if it exists.</p>   <p><strong class="example">Example 1:</strong></p> <img alt="" src="https://assets.leetcode.com/uploads/2023/03/29/example.png" style="width: 241px;   <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      Grid is made up of x connected components, where 1 connected component is made up of  water cells connected together. Fisher can start at any cell within a component and get all fish within it. So, find component with max fish in the grid.

We can do bfs/dfs starting from water cell. And as we traverse the watercell, we can update it to a land cell, after counting the fish from it. This way original grid can act as visited array. Visited condition is if a cell is 0, then it is visited/unreachable.
    </Card>

```ruby title="Maximum Number of Fish in a Grid" lines
# @param {Integer[][]} grid
# @return {Integer}
def find_max_fish(grid)
  m = grid.size; n = grid[0].size
  directions = [[0,1],[0,-1],[1,0],[-1,0]]
  # visted condition is grid[i][j]==0, then consider it visited
  
  neighbors=->(x,y) {
    directions.map {|dx,dy| 
      ((x+dx).between?(0,m-1) && (y+dy).between?(0,n-1) && grid[x+dx][y+dy]!=0) ? [x+dx,y+dy]: nil
    }.compact
  }

  bfs=->(x,y) {
    fish_count = 0
    queue = [[x,y]]

    until queue.empty?
      r,c = queue.pop
      next if grid[r][c]==0 # already visited
      fish_count+=grid[r][c]
      grid[r][c] = 0 # update visited property after counting the fish in this cell
      neighbors.call(r,c).each { |n_r, n_c| queue << [n_r,n_c] }
    end
    fish_count
  }
  max_fish_count = 0

  (0...m).each do |i| 
    (0...n).each do |j| 
      if grid[i][j]!=0
        component_fish_count = bfs.call(i,j) 
        max_fish_count = [max_fish_count, component_fish_count].max
      end
    end
  end

  max_fish_count
end
```
