    <Card>
      <p>Given an <code>m x n</code> matrix <code>mat</code> where every row is sorted in <strong>strictly</strong> <strong>increasing</strong> order, return <em>the <strong>smallest common element</strong> in all rows</em>.</p>  <p>If there is no common element, return <code>-1</code>.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> mat = [[1,2,3,4,5],[2,4,5,8,10],[3,5,7,9,11],[1,3,5,7,9]] <br/> <strong>Output:</strong> 5   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> mat = [[1,2,3],[2,3,4],[2,3,5]] <br/> <strong>Output:</strong> 2    <p><strong>Constraints:</strong></p>  <ul>  <li><code>m == mat.length</code></li>  <li><code>n == mat[i].length</code></li>  <li><code>1 &lt;= m, n &lt;= 500</code></li>  <li><code>1 &lt;= mat[i][j] &lt;= 10<sup>4</sup></code></li>  <li><code>mat[i]</code> is sorted in strictly increasing order.</li> </ul> 
    </Card>

  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      Each row has distinct numbers, so for a number to be present in all rows, its freq should be equal to row count.

Now problem is just about freq, and checking from smallest to largest.
</Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Find Smallest Common Element in All Rows" lines
# @param {Integer[][]} mat
# @return {Integer}
def smallest_common_element(mat)
  mat.flatten.tally.sort.detect {|k,v| v==mat.size }&.first || - 1
end
```
