Therefore, the answer is 4.  <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> words = [&quot;pa&quot;,&quot;papa&quot;,&quot;ma&quot;,&quot;mama&quot;] <br/> <strong>Output:</strong> 2 <br/> <strong>Explanation:</strong> In this example, the counted index pairs are: i = 0 and j = 1 because isPrefixAndSuffix(&quot;pa&quot;, &quot;papa&quot;) is true. i = 2 and j = 3 because isPrefixAndSuffix(&quot;ma&quot;, &quot;mama&quot;) is true. # @param {String[]} words
# @return {Integer}
def count_prefix_suffix_pairs(words)
  words.each_with_index.map do |word, i|
    words[i+1..].count {|x| x.start_with?(word) && x.end_with?(word)}
  end.sum
end
