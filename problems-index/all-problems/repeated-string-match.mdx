      <p>Given two strings <code>a</code> and <code>b</code>, return <em>the minimum number of times you should repeat string </em><code>a</code><em> so that string</em> <code>b</code> <em>is a substring of it</em>. If it is impossible for <code>b</code>â€‹â€‹â€‹â€‹â€‹â€‹ to be a substring of <code>a</code> after repeating it, return <code>-1</code>.</p>  <p><strong>Notice:</strong> string <code>&quot;abc&quot;</code> repeated 0 times is <code>&quot;&quot;</code>, repeated 1 time is <code>&quot;abc&quot;</code> and repeated 2 times is <code>&quot;abcabc&quot;</code>.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> a = &quot;abcd&quot;, b = &quot;cdabcdab&quot; <br/> <strong>Output:</strong> 3 <br/> <strong>Explanation:</strong> We return 3 because by repeating a three times &quot;ab<strong>cdabcdab</strong>cd&quot;, b is a substring of it.   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> a = &quot;a&quot;, b = &quot;aa&quot; <br/> <strong>Output:</strong> 2    <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= a.length, b.length &lt;= 10<sup>4</sup></code></li>  <li><code>a</code> and <code>b</code> consist of lowercase English letters.</li> </ul>

    <Card>
      **Intuition**
See if b is a substring inside of a if a were to be repeated multiple times

**Approach**
1. By just looking at length, see how many times a could fit inside b, call this min_times
2. Do 3 checks, first one for if a was looped min_times times
	 E.g. a = abcd, b = abcd abcd
3. Second one for if a was looped min_times + 1 times, this would catch cases where b started using some ending substring from a, but is not the whole string
    E.g. a = abcd, b = cd abc
4. Third one for if a was looped min_times + 2 times, this would catch cases where b started using some ending substring from a AND where b ended using some beginning substring from a
E.g. a = abcd, b = cd abcd ab

If b is not present in a looped min_times + 2, then there is no match at all
</Card>

```ruby title="Repeated String Match" lines
# @param {String} a
# @param {String} b
# @return {Integer}
# @param {String} a
# @param {String} b
# @return {Integer}
def repeated_string_match(a, b)
  min_times = b.length / a.length

  return min_times if (a * min_times).include?(b)
  return min_times + 1 if (a * (min_times + 1)).include?(b)
  return min_times + 2 if (a * (min_times + 2)).include?(b)
  return -1
end
```
