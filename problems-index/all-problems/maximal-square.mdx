Just calculate size of side of square, considering (i-1,j)  (1,j-1) and (i-1),(j-1) all are 1s it increases by 1, else it is 1, as this cell is a square itself.

so, new value is min of all these other 3 squares  + 1.

```ruby title="Maximal Square" lines
# @param {Character[][]} matrix
# @return {Integer}
def maximal_square(matrix)
  m = matrix.size
  n = matrix[0].size
  
  dp = Array.new(m+1) { Array.new(n+1, 0) }
  max_side = 0

  (1..m).each do |i|
    (1..n).each do |j|
      next if matrix[i-1][j-1] == "0"
      dp[i][j] = 1 + [ dp[i-1][j],dp[i][j-1],dp[i-1][j-1] ].min
      max_side = [max_side, dp[i][j]].max
    end
  end

  max_side * max_side
end
```
