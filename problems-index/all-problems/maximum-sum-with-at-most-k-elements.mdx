The task is to find the <strong>maximum sum</strong> of <strong>at most</strong> <code>k</code> elements from the matrix <code>grid</code> such that:</p>  <ul data-spread="false">  <li>  <p>The number of elements taken from the <code>i<sup>th</sup></code> row of <code>grid</code> does not exceed <code>limits[i]</code>.</p>  </li> </ul>  <p data-pm-slice="1 1 []">Return the <strong>maximum sum</strong>.</p>   <p><strong class="example">Example 1:</strong></p>  <div class="example-block"> <p><strong>Input:</strong> <span class="example-io">grid = [[1,2],[3,4]], limits = [1,2], k = 2</span></p>  <p><br/> <strong>Output:</strong> <span class="example-io">7</span></p>  <p><br/> <strong>Explanation:</strong></p>  <ul>  <li>From the second row, we can take at most 2 elements. The elements taken are 4 and 3.</li>  <li>The maximum possible sum of at most 2 selected elements is <code>4 + 3 = 7</code>.</li> </ul> </div>  <p><strong class="example">Example 2:</strong></p>  <div class="example-block"> <p><strong>Input:</strong> <span class="example-io">grid = [[5,3,7],[8,2,6]], limits = [2,2], k = 3</span></p>  <p><br/> <strong>Output:</strong> <span class="example-io">21</span></p>  <p><br/> <strong>Explanation:</strong></p>  <ul>  <li>From the first row, we can take at most 2 elements.   <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      Just simulate, put [num, row_idx] into pq. Pop max element, if limit is allowed for its row, use it, and reduce limit.
</Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```python3 title="Maximum Sum With at Most K Elements" lines
class Solution:
    def maxSum(self, grid: List[List[int]], limits: List[int], k: int) -> int:
        pq = []
        n = len(grid)
        
        # Add all elements to the heap with their row indices
        for i in range(n):
            for x in grid[i]:
                # Negate x for max heap simulation
                heapq.heappush(pq, (-x, i))
        
        sum = 0
        num_count = 0
        
        while pq:
            if num_count == k:
                return sum
                
            # Get the largest value (negated) and its row index
            neg_num, idx = heapq.heappop(pq)
            num = -neg_num  # Convert back to positive
            
            if limits[idx] > 0:
                sum += num
                num_count += 1
                limits[idx] -= 1
                
        return sum
```
