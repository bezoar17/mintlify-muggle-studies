      <p>You are given the array <code>nums</code> consisting of <code>n</code> positive integers. You computed the sum of all non-empty continuous subarrays from the array and then sorted them in non-decreasing order, creating a new array of <code>n * (n + 1) / 2</code> numbers.</p>  <p><em>Return the sum of the numbers from index </em><code>left</code><em> to index </em><code>right</code> (<strong>indexed from 1</strong>)<em>, inclusive, in the new array. </em>Since the answer can be a huge number return it modulo <code>10<sup>9</sup> + 7</code>.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> nums = [1,2,3,4], n = 4, left = 1, right = 5 <br/> <strong>Output:</strong> 13  <br/> <strong>Explanation:</strong> All subarray sums are 1, 3, 6, 10, 2, 5, 9, 3, 7, 4. After sorting them in non-decreasing order we have the new array [1, 2, 3, 3, 4, 5, 6, 7, 9, 10]. The sum of the numbers from index le = 1 to ri = 5 is 1 + 2 + 3 + 3 + 4 = 13.    <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> nums = [1,2,3,4], n = 4, left = 3, right = 4 <br/> <strong>Output:</strong> 6 <br/> <strong>Explanation:</strong> The given array is the same as example 1. The sum of the numbers from index le = 3 to ri = 4 is 3 + 3 = 6.   <p><strong class="example">Example 3:</strong></p>   <strong>Input:</strong> nums = [1,2,3,4], n = 4, left = 1, right = 10 <br/> <strong>Output:</strong> 50    <p><strong>Constraints:</strong></p>  <ul>  <li><code>n == nums.length</code></li>  <li><code>1 &lt;= nums.length &lt;= 1000</code></li>  <li><code>1 &lt;= nums[i] &lt;= 100</code></li>  <li><code>1 &lt;= left &lt;= right &lt;= n * (n + 1) / 2</code></li> </ul>

  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      Just use prefix sum to calculate sum of sub-array in O(1). Then do same after sorting on all subarrays sum.
</Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Range Sum of Sorted Subarray Sums" lines
# @param {Integer[]} nums
# @param {Integer} n
# @param {Integer} left
# @param {Integer} right
# @return {Integer}
# Read-Only for queries of sum of [l,r]
# pseudo_tree.query(l, r+1)

class PseudoSumSegTree
  def initialize(arr)
    @n = arr.size # size of the source array
    # query can have r till n, so sum of all elements is at prefix[n]
    @prefix_sum_arr = (1..@n).reduce([0]) {|res, i| res << res.last + arr[i-1]; res } # sum till i-1
  end
  
  # [l,r)
  def query(l,r)
    @prefix_sum_arr[r] - @prefix_sum_arr[l]
  end
end

def range_sum(nums, n, left, right)
  sb_sums = []; 
  psg = PseudoSumSegTree.new(nums)
  (0...n).each do |i|
    (i...n).each do |j|
      sb_sums << psg.query(i,j+1)
    end
  end
  sb_psg = PseudoSumSegTree.new(sb_sums.sort)
  sb_psg.query(left-1, right) % (1_000_000_000 + 7)
end
```
