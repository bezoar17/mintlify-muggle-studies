Each element must have the <b>same probability</b> of being returned.</li> </ul>  <p>You must implement the functions of the class such that each function works in&nbsp;<strong>average</strong>&nbsp;<code>O(1)</code>&nbsp;time complexity.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input</strong> [&quot;RandomizedSet&quot;, &quot;insert&quot;, &quot;remove&quot;, &quot;insert&quot;, &quot;getRandom&quot;, &quot;remove&quot;, &quot;insert&quot;, &quot;getRandom&quot;] [[], [1], [2], [2], [], [1], [2], []] <strong>Output</strong> [null, true, false, true, 2, true, false, 2]  <strong>Explanation</strong> RandomizedSet randomizedSet = new RandomizedSet(); randomizedSet.insert(1); // Inserts 1 to the set.     <Card>
      Keep values in array/list, and always delete from last, so thats O(1), getting random also becomes easy as its in array. To always delete from last, effectively swap elements before deletion, and store value to actual idx in a separate hash.

Maintain an array of values, and hash with index positions. Insert is easy at end of array. Delete should also be at end of array, so first swap the values using the index from the hash, to move the value to be deleted to the end, and then delete it, this will be O(1) in a list, effectively, and update the size. To get random, just use inbuilt function.

```ruby title="Insert Delete GetRandom O(1)" lines
class RandomizedSet
    def initialize()
      @size = 0
      @hash = Hash.new
      @arr = Array.new
    end


=begin
    :type val: Integer
    :rtype: Boolean
=end
    def insert(val)
      return false if @hash[val]
      @hash[val] = @size
      @arr[@size] = val
      @size += 1
      true
    end


=begin
    :type val: Integer
    :rtype: Boolean
=end
    def remove(val)
      del_idx = @hash.delete(val)
      return false unless del_idx
      @size-=1

      unless del_idx == @size # already at last
        last = @arr[@size]
        @arr[del_idx] = last
        @hash[last] = del_idx
      end
      true
    end


=begin
    :rtype: Integer
=end
    def get_random()
      @arr[rand(@size)]
    end


end

# Your RandomizedSet object will be instantiated and called as such:
# obj = RandomizedSet.new()
# param_1 = obj.insert(val)
# param_2 = obj.remove(val)
# param_3 = obj.get_random()
```
