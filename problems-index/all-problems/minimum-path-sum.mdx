      <p>Given a <code>m x n</code> <code>grid</code> filled with non-negative numbers, find a path from top left to bottom right, which minimizes the sum of all numbers along its path.</p>  <p><strong>Note:</strong> You can only move either down or right at any point in time.</p>   <p><strong class="example">Example 1:</strong></p> <img alt="" src="https://assets.leetcode.com/uploads/2020/11/05/minpath.jpg" style="width: 242px; height: 242px;" />  <strong>Input:</strong> grid = [[1,3,1],[1,5,1],[4,2,1]] <br/> <strong>Output:</strong> 7 <br/> <strong>Explanation:</strong> Because the path 1 &rarr; 3 &rarr; 1 &rarr; 1 &rarr; 1 minimizes the sum.   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> grid = [[1,2,3],[4,5,6]] <br/> <strong>Output:</strong> 12    <p><strong>Constraints:</strong></p>  <ul>  <li><code>m == grid.length</code></li>  <li><code>n == grid[i].length</code></li>  <li><code>1 &lt;= m, n &lt;= 200</code></li>  <li><code>0 &lt;= grid[i][j] &lt;= 200</code></li> </ul>

    <Card>
      calculate minimum for a given i,j

You can only arrive via i-1, j or i, j-1. 

Choose minimum of these 2 and store this value. 

Initialize the first row and column, as they have only 1 option.

if we go i,j from 0,0 to m-1,n-1. To calculate current value all previous values would have been calculated.
</Card>

```ruby title="Minimum Path Sum" lines
# @param {Integer[][]} grid
# @return {Integer}
def min_path_sum(grid)
  m = grid.size
  n = grid[0].size
  dp = Array.new(m) { Array.new(n, -1) }
  dp[0][0] = grid[0][0]

  (1...n).each { |j| dp[0][j] = dp[0][j-1] + grid[0][j] }
  (1...m).each { |j| dp[j][0] = dp[j-1][0] + grid[j][0] }

  (1...m).each do |i|
    (1...n).each do |j|
      dp[i][j] = grid[i][j] + [dp[i-1][j], dp[i][j-1]].min
    end
  end

  dp[m-1][n-1]
end
```
