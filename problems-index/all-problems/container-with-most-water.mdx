      <p>You are given an integer array <code>height</code> of length <code>n</code>. There are <code>n</code> vertical lines drawn such that the two endpoints of the <code>i<sup>th</sup></code> line are <code>(i, 0)</code> and <code>(i, height[i])</code>.</p>  <p>Find two lines that together with the x-axis form a container, such that the container contains the most water.</p>  <p>Return <em>the maximum amount of water a container can store</em>.</p>  <p><strong>Notice</strong> that you may not slant the container.</p>   <p><strong class="example">Example 1:</strong></p> <img alt="" src="https://s3-lc-upload.s3.amazonaws.com/uploads/2018/07/17/question_11.jpg" style="width: 600px; height: 287px;" />  <strong>Input:</strong> height = [1,8,6,2,5,4,8,3,7] <br/> <strong>Output:</strong> 49 <br/> <strong>Explanation:</strong> The above vertical lines are represented by array [1,8,6,2,5,4,8,3,7].     <Card>
      Similar to sum of 2 values in ordered array to target.

Each step calculate area, to maximize area min(a,b) needs to be increased, so move inwards from the element which is lower
    </Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Container With Most Water" lines
# @param {Integer[]} height
# @return {Integer}
def max_area(height)
  left_p = 0; right_p = height.size - 1;
  max_area = 0
  while right_p > left_p
    area = (right_p - left_p) * [height[left_p], height[right_p]].min;
    max_area = [area, max_area].max

    if height[left_p] < height[right_p]
      left_p+=1
    else
      right_p-=1
    end
  end
  max_area
end
```
