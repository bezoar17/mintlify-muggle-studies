      <p>Alice has some number of cards and she wants to rearrange the cards into groups so that each group is of size <code>groupSize</code>, and consists of <code>groupSize</code> consecutive cards.</p>  <p>Given an integer array <code>hand</code> where <code>hand[i]</code> is the value written on the <code>i<sup>th</sup></code> card and an integer <code>groupSize</code>, return <code>true</code> if she can rearrange the cards, or <code>false</code> otherwise.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> hand = [1,2,3,6,2,3,4,7,8], groupSize = 3 <br/> <strong>Output:</strong> true <br/> <strong>Explanation:</strong> Alice&#39;s hand can be rearranged as [1,2,3],[2,3,4],[6,7,8]   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> hand = [1,2,3,4,5], groupSize = 4 <br/> <strong>Output:</strong> false <br/> <strong>Explanation:</strong> Alice&#39;s hand can not be rearranged into groups of 4.     <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= hand.length &lt;= 10<sup>4</sup></code></li>  <li><code>0 &lt;= hand[i] &lt;= 10<sup>9</sup></code></li>  <li><code>1 &lt;= groupSize &lt;= hand.length</code></li> </ul>   <p><strong>Note:</strong> This question is the same as 1296: <a href="https://leetcode.com/problems/divide-array-in-sets-of-k-consecutive-numbers/" target="_blank">https://leetcode.com/problems/divide-array-in-sets-of-k-consecutive-numbers/</a></p>

# @param {Integer[]} hand
# @param {Integer} group_size
# @return {Boolean}
def is_n_straight_hand(hand, group_size)
  n = hand.size
  return false if n%group_size != 0

  count_h = hand.sort.tally

  (n/group_size).times do |i|
    starting_key = count_h.first[0]
    group_size.times {
      return false unless count_h.has_key?(starting_key)
      count_h[starting_key] -=1
      starting_key+=1
    }
    count_h.delete_if {|k,v| v==0}
  end
  true
end
