      <p>A <strong>wiggle sequence</strong> is a sequence where the differences between successive numbers strictly alternate between positive and negative. The first difference (if one exists) may be either positive or negative. A sequence with one element and a sequence with two non-equal elements are trivially wiggle sequences.</p>  <ul>  <li>For example, <code>[1, 7, 4, 9, 2, 5]</code> is a <strong>wiggle sequence</strong> because the differences <code>(6, -3, 5, -7, 3)</code> alternate between positive and negative.</li>  <li>In contrast, <code>[1, 4, 7, 2, 5]</code> and <code>[1, 7, 4, 5, 5]</code> are not wiggle sequences. The first is not because its first two differences are positive, and the second is not because its last difference is zero.</li> </ul>  <p>A <strong>subsequence</strong> is obtained by deleting some elements (possibly zero) from the original sequence, leaving the remaining elements in their original order.</p>  <p>Given an integer array <code>nums</code>, return <em>the length of the longest <strong>wiggle subsequence</strong> of </em><code>nums</code>.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> nums = [1,7,4,9,2,5] <br/> <strong>Output:</strong> 6 <br/> <strong>Explanation:</strong> The entire sequence is a wiggle sequence with differences (6, -3, 5, -7, 3).   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> nums = [1,17,5,10,13,15,10,5,16,8] <br/> <strong>Output:</strong> 7 <br/> <strong>Explanation:</strong> There are several subsequences that achieve this length. One is [1, 17, 10, 13, 10, 16, 8] with differences (16, -7, 3, -3, 6, -8).   <p><strong class="example">Example 3:</strong></p>   <strong>Input:</strong> nums = [1,2,3,4,5,6,7,8,9] <br/> <strong>Output:</strong> 2    <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= nums.length &lt;= 1000</code></li>  <li><code>0 &lt;= nums[i] &lt;= 1000</code></li> </ul>   <p><strong>Follow up:</strong> Could you solve this in <code>O(n)</code> time?</p>

class Solution {
public:
    
	int wiggleMaxLength(vector<int>& nums) {
        
        if(nums.size()<2)
        return nums.size();
        int res=nums[1]==nums[0] ? 1:2;
        int prevdiff=nums[1]-nums[0];
        for(int i=2;i<nums.size();i++){
        	int diff=nums[i]-nums[i-1];
        	if((diff>0 && prevdiff<=0) || (diff<0 && prevdiff>=0)){
        		res++;
        	prevdiff=diff;
        	}
        }
        return res;
    }
};
