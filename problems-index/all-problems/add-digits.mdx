      <p>Given an integer <code>num</code>, repeatedly add all its digits until the result has only one digit, and return it.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> num = 38 <br/> <strong>Output:</strong> 2 <br/> <strong>Explanation:</strong> The process is 38 --&gt; 3 + 8 --&gt; 11 11 --&gt; 1 + 1 --&gt; 2  Since 2 has only one digit, return it.   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> num = 0 <br/> <strong>Output:</strong> 0    <p><strong>Constraints:</strong></p>  <ul>  <li><code>0 &lt;= num &lt;= 2<sup>31</sup> - 1</code></li> </ul>   <p><strong>Follow up:</strong> Could you do it without any loop/recursion in <code>O(1)</code> runtime?</p>

class Solution {
public:
    int addDigits(int num) {
        if(num==0) return 0;
        else
        return (num-1)%9+1;
    }
};
