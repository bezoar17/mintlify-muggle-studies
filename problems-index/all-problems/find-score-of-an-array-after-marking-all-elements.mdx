      <p>You are given an array <code>nums</code> consisting of positive integers.</p>  <p>Starting with <code>score = 0</code>, apply the following algorithm:</p>  <ul>  <li>Choose the smallest integer of the array that is not marked. If there is a tie, choose the one with the smallest index.</li>  <li>Add the value of the chosen integer to <code>score</code>.</li>  <li>Mark <strong>the chosen element and its two adjacent elements if they exist</strong>.</li>  <li>Repeat until all the array elements are marked.</li> </ul>  <p>Return <em>the score you get after applying the above algorithm</em>.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> nums = [2,1,3,4,5,2] <br/> <strong>Output:</strong> 7 <br/> <strong>Explanation:</strong> We mark the elements as follows: - 1 is the smallest unmarked element, so we mark it and its two adjacent elements: [<u>2</u>,<u>1</u>,<u>3</u>,4,5,2]. - 2 is the smallest unmarked element, so we mark it and its left adjacent element: [<u>2</u>,<u>1</u>,<u>3</u>,4,<u>5</u>,<u>2</u>]. - 4 is the only remaining unmarked element, so we mark it: [<u>2</u>,<u>1</u>,<u>3</u>,<u>4</u>,<u>5</u>,<u>2</u>]. Our score is 1 + 2 + 4 = 7.   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> nums = [2,3,5,1,3,2] <br/> <strong>Output:</strong> 5 <br/> <strong>Explanation:</strong> We mark the elements as follows: - 1 is the smallest unmarked element, so we mark it and its two adjacent elements: [2,3,<u>5</u>,<u>1</u>,<u>3</u>,2]. - 2 is the smallest unmarked element, since there are two of them, we choose the left-most one, so we mark the one at index 0 and its right adjacent element: [<u>2</u>,<u>3</u>,<u>5</u>,<u>1</u>,<u>3</u>,2]. - 2 is the only remaining unmarked element, so we mark it: [<u>2</u>,<u>3</u>,<u>5</u>,<u>1</u>,<u>3</u>,<u>2</u>]. Our score is 1 + 2 + 2 = 5.    <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= nums.length &lt;= 10<sup>5</sup></code></li>  <li><code>1 &lt;= nums[i] &lt;= 10<sup>6</sup></code></li> </ul>

```python3 title="Find Score of an Array After Marking All Elements" lines
import heapq

class Solution:
    def findScore(self, nums: List[int]) -> int:
        pq = [(num, i) for i, num in enumerate(nums)]
        heapq.heapify(pq)
        
        score = 0
        visited = set()
        
        while pq:
            num, idx = heapq.heappop(pq)            
            if idx in visited:
                continue
            
            score += abs(num)
            visited.add(idx)
            visited.add(idx + 1)
            visited.add(idx - 1)
        
        return score
```
