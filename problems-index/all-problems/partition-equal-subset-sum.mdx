    <Card>
      <p>Given an integer array <code>nums</code>, return <code>true</code> <em>if you can partition the array into two subsets such that the sum of the elements in both subsets is equal or </em><code>false</code><em> otherwise</em>.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> nums = [1,5,11,5] <br/> <strong>Output:</strong> true <br/> <strong>Explanation:</strong> The array can be partitioned as [1, 5, 5] and [11].   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> nums = [1,2,3,5] <br/> <strong>Output:</strong> false <br/> <strong>Explanation:</strong> The array cannot be partitioned into equal sum subsets.    <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= nums.length &lt;= 200</code></li>  <li><code>1 &lt;= nums[i] &lt;= 100</code></li> </ul>

  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      Subset generation using [] as starting set, and each element doubles the different subsets. Subsets merge into smaller space, here due to different sums. Other problem is https://leetcode.com/problems/smallest-sufficient-team/.

    </Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Partition Equal Subset Sum" lines
def can_partition(nums)
  nums = nums
  return false unless nums.sum.even?
  target = nums.sum / 2
  
  partial_sums = Set[0]

  nums.each do |num|
    new_partial_sums = Set[]

    partial_sums.each do |partial|
      new_partial = partial + num
      return true if new_partial == target
      next if new_partial > target

      new_partial_sums.add(new_partial)
    end

    partial_sums.merge(
      new_partial_sums
    )
  end

  false
end
```
