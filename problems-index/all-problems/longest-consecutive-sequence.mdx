      <p>Given an unsorted array of integers <code>nums</code>, return <em>the length of the longest consecutive elements sequence.</em></p>  <p>You must write an algorithm that runs in&nbsp;<code>O(n)</code>&nbsp;time.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> nums = [100,4,200,1,3,2] <br/> <strong>Output:</strong> 4 <br/> <strong>Explanation:</strong> The longest consecutive elements sequence is <code>[1, 2, 3, 4]</code>. Therefore its length is 4.   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> nums = [0,3,7,2,5,8,4,6,0,1] <br/> <strong>Output:</strong> 9   <p><strong class="example">Example 3:</strong></p>   <strong>Input:</strong> nums = [1,0,1,2] <br/> <strong>Output:</strong> 3    <p><strong>Constraints:</strong></p>  <ul>  <li><code>0 &lt;= nums.length &lt;= 10<sup>5</sup></code></li>  <li><code>-10<sup>9</sup> &lt;= nums[i] &lt;= 10<sup>9</sup></code></li> </ul>

      For O(n), we have to process elements once or twice only.

Once we store a set of all no.s, we can check for consecutive in O(n)

    </Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Longest Consecutive Sequence" lines
# @param {Integer[]} nums
# @return {Integer}
def longest_consecutive(nums)
  s_ = nums.to_set
  max_count = 0
  
  nums.each do |num|
    if !s_.include?(num-1) # start of a sequence
      counter = 0; temp = num
      while s_.include?(temp)
        counter+=1
        temp+=1
      end
      max_count = [counter, max_count].max
    end
  end

  max_count
end
```
