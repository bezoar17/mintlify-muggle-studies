      <p>Given an array of strings <code>words</code> and a width <code>maxWidth</code>, format the text such that each line has exactly <code>maxWidth</code> characters and is fully (left and right) justified.</p>  <p>You should pack your words in a greedy approach; that is, pack as many words as you can in each line. Pad extra spaces <code>&#39; &#39;</code> when necessary so that each line has exactly <code>maxWidth</code> characters.</p>  <p>Extra spaces between words should be distributed as evenly as possible. If the number of spaces on a line does not divide evenly between words, the empty slots on the left will be assigned more spaces than the slots on the right.</p>  <p>For the last line of text, it should be left-justified, and no extra space is inserted between words.</p>  <p><strong>Note:</strong></p>  <ul>  <li>A word is defined as a character sequence consisting of non-space characters only.</li>  <li>Each word&#39;s length is guaranteed to be greater than <code>0</code> and not exceed <code>maxWidth</code>.</li>  <li>The input array <code>words</code> contains at least one word.</li> </ul>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> words = [&quot;This&quot;, &quot;is&quot;, &quot;an&quot;, &quot;example&quot;, &quot;of&quot;, &quot;text&quot;, &quot;justification.&quot;], maxWidth = 16 <br/> <strong>Output:</strong> [ &nbsp; &nbsp;&quot;This &nbsp; &nbsp;is &nbsp; &nbsp;an&quot;, &nbsp; &nbsp;&quot;example &nbsp;of text&quot;, &nbsp; &nbsp;&quot;justification. &nbsp;&quot; ]  <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> words = [&quot;What&quot;,&quot;must&quot;,&quot;be&quot;,&quot;acknowledgment&quot;,&quot;shall&quot;,&quot;be&quot;], maxWidth = 16 <br/> <strong>Output:</strong> [ &nbsp; &quot;What &nbsp; must &nbsp; be&quot;, &nbsp; &quot;acknowledgment &nbsp;&quot;, &nbsp; &quot;shall be &nbsp; &nbsp; &nbsp; &nbsp;&quot; ] <br/> <strong>Explanation:</strong> Note that the last line is &quot;shall be    &quot; instead of &quot;shall     be&quot;, because the last line must be left-justified instead of fully-justified. Note that the second line is also left-justified because it contains only one word.  <p><strong class="example">Example 3:</strong></p>   <strong>Input:</strong> words = [&quot;Science&quot;,&quot;is&quot;,&quot;what&quot;,&quot;we&quot;,&quot;understand&quot;,&quot;well&quot;,&quot;enough&quot;,&quot;to&quot;,&quot;explain&quot;,&quot;to&quot;,&quot;a&quot;,&quot;computer.&quot;,&quot;Art&quot;,&quot;is&quot;,&quot;everything&quot;,&quot;else&quot;,&quot;we&quot;,&quot;do&quot;], maxWidth = 20 <br/> <strong>Output:</strong> [ &nbsp; &quot;Science &nbsp;is &nbsp;what we&quot;,   &quot;understand &nbsp; &nbsp; &nbsp;well&quot;, &nbsp; &quot;enough to explain to&quot;, &nbsp; &quot;a &nbsp;computer. &nbsp;Art is&quot;, &nbsp; &quot;everything &nbsp;else &nbsp;we&quot;, &nbsp; &quot;do &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&quot; ]   <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= words.length &lt;= 300</code></li>  <li><code>1 &lt;= words[i].length &lt;= 20</code></li>  <li><code>words[i]</code> consists of only English letters and symbols.</li>  <li><code>1 &lt;= maxWidth &lt;= 100</code></li>  <li><code>words[i].length &lt;= maxWidth</code></li> </ul>

# @param {String[]} words
# @param {Integer} max_width
# @return {String[]}

Line = Struct.new(:words, :mw) do
  def can_fit?(word)
    remaining_space = mw - words.map(&:size).sum - words.size
    word.size <= remaining_space
  end

  # justify each line
  # assigning space is also straightforward, for each line
  # evenly, if not even, then more on left slot than right
  def justify(left_only = false)
    total_space_count = mw - words.map(&:size).sum
    return words.first if words.size == 1 && total_space_count == 0
    
    # non-zero space
    regions = [1, words.size - 1].max # atleast 1 space region
    
    per_region_space_count = left_only ? 1 : (total_space_count/regions.to_f).floor
    space_dist = [per_region_space_count] * regions

    extra_spaces = total_space_count - space_dist.sum
    
    unless left_only # full justification, assign extras to left first
      i = 0
      (space_dist[i]+=1; extra_spaces-=1; i+=1) while extra_spaces >= 1
    end

    output = (0...words.size).map { |idx| words[idx] + " " * (space_dist[idx].to_i) }.join
    output + " "*extra_spaces
  end
end

def full_justify(words, max_width)
  # assigning which words go on which line, is straightforward
  lines = [Line.new([], max_width)]
  
  words.each do |word|
    curr_line = lines.last
    if curr_line.can_fit?(word)
      curr_line.words.append(word)
    else
      lines << Line.new([word], max_width)
    end
  end

  lines[..-2].map { |l| l.justify } + [lines.last.justify(true)]
end
