  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      Calculate sum of each node, considering it as root in a dfs way.

If ever we see total_sum/2 in a subtree-sum it can be done, else it can't.
</Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Equal Tree Partition" lines
# Definition for a binary tree node.
# class TreeNode
#     attr_accessor :val, :left, :right
#     def initialize(val = 0, left = nil, right = nil)
#         @val = val
#         @left = left
#         @right = right
#     end
# end
# @param {TreeNode} root
# @return {Boolean}
def check_equal_tree(root)
  return false unless root

  @root = root
  @all_sums = []
  @total_sum = tree_sum(root)
  @all_sums.include?(@total_sum/2.0)
end

def tree_sum(node)
  return 0 unless node

  sum = node.val + tree_sum(node.left) + tree_sum(node.right)
  @all_sums << sum.to_f if node!=@root
  sum
end
```
