    <Card>
      <p>Given an array of integers <code>arr</code>, return <em>the number of subarrays with an <strong>odd</strong> sum</em>.</p>  <p>Since the answer can be very large, return it modulo <code>10<sup>9</sup> + 7</code>.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> arr = [1,3,5] <br/> <strong>Output:</strong> 4 <br/> <strong>Explanation:</strong> All subarrays are [[1],[1,3],[1,3,5],[3],[3,5],[5]] All sub-arrays sum are [1,4,9,3,8,5]. Odd sums are [1,9,3,5] so the answer is 4.   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> arr = [2,4,6] <br/> <strong>Output:</strong> 0 <br/> <strong>Explanation:</strong> All subarrays are [[2],[2,4],[2,4,6],[4],[4,6],[6]] All sub-arrays sum are [2,6,12,4,10,6]. All sub-arrays have even sum and the answer is 0.   <p><strong class="example">Example 3:</strong></p>   <strong>Input:</strong> arr = [1,2,3,4,5,6,7] <br/> <strong>Output:</strong> 16    <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= arr.length &lt;= 10<sup>5</sup></code></li>  <li><code>1 &lt;= arr[i] &lt;= 100</code></li> </ul>

  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      Calculate running prefix_sum for each i. And also no. of previous idxs where we had even sums and odd sums. If at an i, we have current_sum is odd, all previous even sum ending values can be starting points with i as end, so we add that. And vice versa.
    </Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Number of Sub-arrays With Odd Sum" lines
# @param {Integer[]} arr
# @return {Integer}
def num_of_subarrays(arr)
  big_mod = 1_000_000_000 + 7
  current_sum = 0
  even_sum = 1
  odd_sum = 0
  sb_count = 0
  
  # i is end of sb_arr, if end is odd, even sum ones can be start points and vice-versa
  arr.each do |num|
    current_sum+=num
    if current_sum.odd?
      sb_count+=even_sum
      odd_sum+=1
    else
      sb_count+=odd_sum
      even_sum+=1
    end
  end

  sb_count % big_mod
end
```
