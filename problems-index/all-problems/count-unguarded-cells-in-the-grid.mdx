      <p>You are given two integers <code>m</code> and <code>n</code> representing a <strong>0-indexed</strong> <code>m x n</code> grid. You are also given two 2D integer arrays <code>guards</code> and <code>walls</code> where <code>guards[i] = [row<sub>i</sub>, col<sub>i</sub>]</code> and <code>walls[j] = [row<sub>j</sub>, col<sub>j</sub>]</code> represent the positions of the <code>i<sup>th</sup></code> guard and <code>j<sup>th</sup></code> wall respectively.</p>  <p>A guard can see <b>every</b> cell in the four cardinal directions (north, east, south, or west) starting from their position unless <strong>obstructed</strong> by a wall or another guard. A cell is <strong>guarded</strong> if there is <strong>at least</strong> one guard that can see it.</p>  <p>Return<em> the number of unoccupied cells that are <strong>not</strong> <strong>guarded</strong>.</em></p>   <p><strong class="example">Example 1:</strong></p> <img alt="" src="https://assets.leetcode.com/uploads/2022/03/10/example1drawio2.png" style="width: 300px; height: 204px;" />  <strong>Input:</strong> m = 4, n = 6, guards = [[0,0],[1,1],[2,3]], walls = [[0,1],[2,2],[1,4]] <br/> <strong>Output:</strong> 7 <br/> <strong>Explanation:</strong> The guarded and unguarded cells are shown in red and green respectively in the above diagram.     <Card>
      Initialize a grid, and mark walls and guards. For each guard and direction, travel and mark guarded cells. Stop when you see another guard or wall. We can use 1 for unguarded cell, initially all will be 1.
0 for wall, -1 for a guard, and -2 for guarded cell. Start from -1 cells, move and mark cells -2, stop when you see -1or 0.
</Card>

```ruby title="Count Unguarded Cells in the Grid" lines
# @param {Integer} m
# @param {Integer} n
# @param {Integer[][]} guards
# @param {Integer[][]} walls
# @return {Integer}
def count_unguarded(m, n, guards, walls)
  grid = Array.new(m) { Array.new(n, 1) }
  walls.each {|x,y| grid[x][y] = 0 }
  guards.each {|x,y| grid[x][y] = -1}
  directions = [[0,1],[0,-1],[1,0],[-1,0]]
  # 1 is unguarded, -1 is a guard, 0 is a wall, -2 is guarded cell

  travel=->(i,j,dir_idx) {
    di,dj = directions[dir_idx]
    n_i = i+di; n_j= j+dj
    if n_i.between?(0,m-1) && n_j.between?(0,n-1) && grid[n_i][n_j]!=0 && grid[n_i][n_j]!=-1
      grid[n_i][n_j]=-2; travel.call(n_i,n_j,dir_idx)
    end
  }

  # for each guard and direction mark guarded cells
  guards.each { |g| (0..3).each {|dir_idx| travel.call(g[0],g[1], dir_idx)} }
  grid.map {|row| row.count{|x| x>0} }.sum
end
```
