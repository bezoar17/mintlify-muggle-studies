    <Card>
      * Serialization-Deserialzation of tree

Classic, do a pre-order with all nodes separated by a ,. And record all nil values, as null.

While de-serializing, pick the top always, and return nil if its null.
Else make it root, and continue with left, right as in pre-order, always picking top of nodes array.
</Card>

```ruby title="Serialize and Deserialize Binary Tree" lines
# Definition for a binary tree node.
# class TreeNode
#     attr_accessor :val, :left, :right
#     def initialize(val)
#         @val = val
#         @left, @right = nil, nil
#     end
# end

# Encodes a tree to a single string.
#
# @param {TreeNode} root
# @return {string}

def serialize(root)
  recurse=->(node, str) {
    return str+="null," if node.nil?

    str+= "#{node.val},"
    str = recurse.call(node.left, str)
    str = recurse.call(node.right, str)
  }
  
  recurse.call(root, '')
end

# Decodes your encoded data to tree.
#
# @param {string} data
# @return {TreeNode}
def deserialize(data)
  nodes = data.split(",")
  
  recurse=->() {
    top_node = nodes.shift
    return nil if top_node == 'null'

    root = TreeNode.new(top_node.to_i)
    root.left = recurse.call
    root.right = recurse.call
    root
  }

  recurse.call
end


# Your functions will be called as such:
# deserialize(serialize(data))
```
