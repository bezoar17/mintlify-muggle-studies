      <p>You are given an array <code>nums</code> consisting of <strong>positive</strong> integers.</p>  <p>We call a subarray of an array <strong>complete</strong> if the following condition is satisfied:</p>  <ul>  <li>The number of <strong>distinct</strong> elements in the subarray is equal to the number of distinct elements in the whole array.</li> </ul>  <p>Return <em>the number of <strong>complete</strong> subarrays</em>.</p>  <p>A <strong>subarray</strong> is a contiguous non-empty part of an array.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> nums = [1,3,1,2,2] <br/> <strong>Output:</strong> 4 <br/> <strong>Explanation:</strong> The complete subarrays are the following: [1,3,1,2], [1,3,1,2,2], [3,1,2] and [3,1,2,2].   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> nums = [5,5,5,5] <br/> <strong>Output:</strong> 10 <br/> <strong>Explanation:</strong> The array consists only of the integer 5, so any subarray is complete. The number of subarrays that we can choose is 10.    <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= nums.length &lt;= 1000</code></li>  <li><code>1 &lt;= nums[i] &lt;= 2000</code></li> </ul>

```ruby title="Count Complete Subarrays in an Array" lines
# @param {Integer[]} nums
# @return {Integer}
def count_complete_subarrays(nums)
  distinct_count = nums.uniq.size
  h = Hash.new(0)
  left = 0
  result = 0
  nums.each_with_index do |num, right|
    h[num] += 1
    while h.size == distinct_count
      result += nums.size - right
      h[nums[left]] -= 1
      h.delete(nums[left]) if h[nums[left]] == 0
      left += 1
    end
  end
  result
end
```
