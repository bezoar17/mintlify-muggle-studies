      <p>Given the <code>head</code> of a linked list, remove the <code>n<sup>th</sup></code> node from the end of the list and return its head.</p>   <p><strong class="example">Example 1:</strong></p> <img alt="" src="https://assets.leetcode.com/uploads/2020/10/03/remove_ex1.jpg" style="width: 542px; height: 222px;" />  <strong>Input:</strong> head = [1,2,3,4,5], n = 2 <br/> <strong>Output:</strong> [1,2,3,5]   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> head = [1], n = 1 <br/> <strong>Output:</strong> []   <p><strong class="example">Example 3:</strong></p>   <strong>Input:</strong> head = [1,2], n = 1 <br/> <strong>Output:</strong> [1]    <p><strong>Constraints:</strong></p>  <ul>  <li>The number of nodes in the list is <code>sz</code>.</li>  <li><code>1 &lt;= sz &lt;= 30</code></li>  <li><code>0 &lt;= Node.val &lt;= 100</code></li>  <li><code>1 &lt;= n &lt;= sz</code></li> </ul>   <p><strong>Follow up:</strong> Could you do this in one pass?</p>

  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      Using a dummy node at start, and removing edge case of 1 node list simplifies the problem greatly.
Move the fast pointer n steps at first, then move both pointers until end, and remove the slow.next node.
    </Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Remove Nth Node From End of List" lines
# Definition for singly-linked list.
# class ListNode
#     attr_accessor :val, :next
#     def initialize(val = 0, _next = nil)
#         @val = val
#         @next = _next
#     end
# end
# @param {ListNode} head
# @param {Integer} n
# @return {ListNode}
def remove_nth_from_end(head, n)
  node_0 = ListNode.new
  node_0.next = head; fast = node_0; slow = node_0
  n.times { fast = fast.next }
  slow, fast = slow.next, fast.next until fast.next.nil?
  slow.next = slow.next.next # slow.next has to be removed
  
  node_0.next
end
```
