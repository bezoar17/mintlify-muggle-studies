      <p>Given a string <code>s</code>.&nbsp;Return&nbsp;all the words vertically in the same order in which they appear in <code>s</code>.<br /> Words are returned as a list of strings, complete with&nbsp;spaces when is necessary. ```ruby title="Print Words Vertically" lines
# @param {String} s
# @return {String[]}
def print_vertically(s)
  words = s.split(" ")
  n = words.size
  max_len = words.map(&:size).max
  all_chars = Array.new(n) { Array.new(max_len," ") }

  words.each_with_index do |word, i|
    word.chars.each_with_index do |ch, j|
      all_chars[i][j] = ch
    end
  end

  all_chars.transpose.map { |row| row.join.rstrip }
end
