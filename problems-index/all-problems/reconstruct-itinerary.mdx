      <p>You are given a list of airline <code>tickets</code> where <code>tickets[i] = [from<sub>i</sub>, to<sub>i</sub>]</code> represent the departure and the arrival airports of one flight. Reconstruct the itinerary in order and return it.</p>  <p>All of the tickets belong to a man who departs from <code>&quot;JFK&quot;</code>, thus, the itinerary must begin with <code>&quot;JFK&quot;</code>. If there are multiple valid itineraries, you should return the itinerary that has the smallest lexical order when read as a single string.</p>  <ul>  <li>For example, the itinerary <code>[&quot;JFK&quot;, &quot;LGA&quot;]</code> has a smaller lexical order than <code>[&quot;JFK&quot;, &quot;LGB&quot;]</code>.</li> </ul>  <p>You may assume all tickets form at least one valid itinerary. You must use all the tickets once and only once.</p>   <p><strong class="example">Example 1:</strong></p> <img alt="" src="https://assets.leetcode.com/uploads/2021/03/14/itinerary1-graph.jpg" style="width: 382px; height: 222px;" />  <strong>Input:</strong> tickets = [[&quot;MUC&quot;,&quot;LHR&quot;],[&quot;JFK&quot;,&quot;MUC&quot;],[&quot;SFO&quot;,&quot;SJC&quot;],[&quot;LHR&quot;,&quot;SFO&quot;]] <br/> <strong>Output:</strong> [&quot;JFK&quot;,&quot;MUC&quot;,&quot;LHR&quot;,&quot;SFO&quot;,&quot;SJC&quot;]   <p><strong class="example">Example 2:</strong></p> <img alt="" src="https://assets.leetcode.com/uploads/2021/03/14/itinerary2-graph.jpg" style="width: 222px; height: 230px;" />  <strong>Input:</strong> tickets = [[&quot;JFK&quot;,&quot;SFO&quot;],[&quot;JFK&quot;,&quot;ATL&quot;],[&quot;SFO&quot;,&quot;ATL&quot;],[&quot;ATL&quot;,&quot;JFK&quot;],[&quot;ATL&quot;,&quot;SFO&quot;]] <br/> <strong>Output:</strong> [&quot;JFK&quot;,&quot;ATL&quot;,&quot;JFK&quot;,&quot;SFO&quot;,&quot;ATL&quot;,&quot;SFO&quot;] <br/> <strong>Explanation:</strong> Another possible reconstruction is [&quot;JFK&quot;,&quot;SFO&quot;,&quot;ATL&quot;,&quot;JFK&quot;,&quot;ATL&quot;,&quot;SFO&quot;] but it is larger in lexical order.    <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= tickets.length &lt;= 300</code></li>  <li><code>tickets[i].length == 2</code></li>  <li><code>from<sub>i</sub>.length == 3</code></li>  <li><code>to<sub>i</sub>.length == 3</code></li>  <li><code>from<sub>i</sub></code> and <code>to<sub>i</sub></code> consist of uppercase English letters.</li>  <li><code>from<sub>i</sub> != to<sub>i</sub></code></li> </ul>

# @param {String[][]} tickets
# @return {String[]}
def find_itinerary(tickets)
  # starting from jfk, the dfs path which visits all edges is the answer
  # in each next path, pick lexicographllicaly smaller edge first
  # here, nodes can be revisited, but edges cant be
  # or, we need a ordering of edges, such that, all can be covered
  return tickets.first if tickets.size == 1
  graph = Hash.new{|h,k| h[k]=[]}
  tickets.each {|u,v| graph[u]<<v}
  graph.each {|k,v| graph[k]=v.sort}
  
  itinerary = []
  stack = ['JFK']
  
  until stack.empty?
    until graph[stack.last].empty?
      stack << graph[stack.last].shift
    end

    itinerary << stack.pop
  end

  itinerary.reverse
end
