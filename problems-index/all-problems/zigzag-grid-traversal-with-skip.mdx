  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      Maintain alternating parity for taking or not taking elements. Also direction for each row is L-R or R-L based on parity as well. Just use these 2 things to build the output.****
    </Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Zigzag Grid Traversal With Skip" lines
# @param {Integer[][]} grid
# @return {Integer[]}
def zigzag_traversal(grid)
  n = grid[0].size
  output = []; take_parity = 0
  grid.each_with_index do |row, row_idx|
    (row_idx.even? ? (0...n) : (n-1).downto(0)).each do |col_idx|
      output << grid[row_idx][col_idx] if take_parity==0
      take_parity = 1-take_parity
    end
  end
  output
end
```
