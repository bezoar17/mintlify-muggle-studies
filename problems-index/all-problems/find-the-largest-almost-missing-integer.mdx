  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      3 cases, k ==1, k==n, and k in (1,n). k ==n, max of nums. k==1, max of nums which appears once. k in (1,n), same logic as k==1, but only first and last element can be these, so check only these.
</Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Find the Largest Almost Missing Integer" lines
# @param {Integer[]} nums
# @param {Integer} k
# @return {Integer}
def largest_integer(nums, k)
  return nums.max if k == nums.size
  tally_h = nums.tally
  return (tally_h.select {|k,v| v==1}.map(&:first).max || -1) if k == 1
  [
    tally_h[nums.first] == 1 ? nums.first : -1,
    tally_h[nums.last] == 1 ? nums.last : -1
  ].max
end
```
