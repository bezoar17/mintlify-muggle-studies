      <p>Given an integer array <code>nums</code>, in which exactly two elements appear only once and all the other elements appear exactly twice. Find the two elements that appear only once. You can return the answer in <strong>any order</strong>.</p>  <p>You must write an&nbsp;algorithm that runs in linear runtime complexity and uses&nbsp;only constant extra space.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> nums = [1,2,1,3,2,5] <br/> <strong>Output:</strong> [3,5] <strong>Explanation: </strong> [5, 3] is also a valid answer.   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> nums = [-1,0] <br/> <strong>Output:</strong> [-1,0]   <p><strong class="example">Example 3:</strong></p>   <strong>Input:</strong> nums = [0,1] <br/> <strong>Output:</strong> [1,0]    <p><strong>Constraints:</strong></p>  <ul>  <li><code>2 &lt;= nums.length &lt;= 3 * 10<sup>4</sup></code></li>  <li><code>-2<sup>31</sup> &lt;= nums[i] &lt;= 2<sup>31</sup> - 1</code></li>  <li>Each integer in <code>nums</code> will appear twice, only two integers will appear once.</li> </ul>

```ruby title="Single Number III" lines
# @param {Integer[]} nums
# @return {Integer[]}
def single_number(nums)
  xor_val = nums.reduce(:^) # a^b, both appear once
  # whereever xor_val has 1, a and b will differ in that bit
  # partition nums by value of that bit, in each partition, now only a or b appear once, others twice
  # so find, 1, and then we have the other
  # here we only keep lsb of xor to partition
  xor_bit = xor_val & -xor_val # keep only lsb
  nums.partition { |n| n & xor_bit > 0 }.map { |arr| arr.reduce(:^)}
end
```
