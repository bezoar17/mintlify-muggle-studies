    <Card>
      <p>Given a&nbsp;string <code>s</code>&nbsp;of zeros and ones, <em>return the maximum score after splitting the string into two <strong>non-empty</strong> substrings</em> (i.e. <strong>left</strong> substring and <strong>right</strong> substring).</p>  <p>The score after splitting a string is the number of <strong>zeros</strong> in the <strong>left</strong> substring plus the number of <strong>ones</strong> in the <strong>right</strong> substring.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> s = &quot;011101&quot; <br/> <strong>Output:</strong> 5  <br/> <strong>Explanation:</strong>  All possible ways of splitting s into two non-empty substrings are: left = &quot;0&quot; and right = &quot;11101&quot;, score = 1 + 4 = 5  left = &quot;01&quot; and right = &quot;1101&quot;, score = 1 + 3 = 4  left = &quot;011&quot; and right = &quot;101&quot;, score = 1 + 2 = 3  left = &quot;0111&quot; and right = &quot;01&quot;, score = 1 + 1 = 2  left = &quot;01110&quot; and right = &quot;1&quot;, score = 2 + 1 = 3   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> s = &quot;00111&quot; <br/> <strong>Output:</strong> 5 <br/> <strong>Explanation:</strong> When left = &quot;00&quot; and right = &quot;111&quot;, we get the maximum score = 2 + 3 = 5   <p><strong class="example">Example 3:</strong></p>   <strong>Input:</strong> s = &quot;1111&quot; <br/> <strong>Output:</strong> 3    <p><strong>Constraints:</strong></p>  <ul>  <li><code>2 &lt;= s.length &lt;= 500</code></li>  <li>The string <code>s</code> consists of characters <code>&#39;0&#39;</code> and <code>&#39;1&#39;</code> only.</li> </ul>

  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      For an idx, calculate prefix_sum, along with total_sum we can get no.of 0 and 1s in both halves. Use it for score, and calculate the max amongst all indices.
    </Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Maximum Score After Splitting a String" lines
# @param {String} s
# @return {Integer}
def max_score(s)
  chars = s.chars.map(&:to_i);
  n = chars.size
  sum = chars.sum
  max_score = 0; count_1 = 0
  chars.each_with_index do |i, idx|
    count_1 += 1 if i == 1
    max_score = [max_score, (idx+1-count_1) + (sum-count_1) ].max if idx!= n-1
  end
  max_score
end
```
