      <p>You are given a <strong>0-indexed</strong> integer array <code>nums</code>. A pair of indices <code>(i, j)</code> is a <strong>bad pair</strong> if <code>i &lt; j</code> and <code>j - i != nums[j] - nums[i]</code>.</p>  <p>Return<em> the total number of <strong>bad pairs</strong> in </em><code>nums</code>.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> nums = [4,1,3,3] <br/> <strong>Output:</strong> 5 <br/> <strong>Explanation:</strong> The pair (0, 1) is a bad pair since 1 - 0 != 1 - 4. The pair (0, 2) is a bad pair since 2 - 0 != 3 - 4, 2 != -1. The pair (0, 3) is a bad pair since 3 - 0 != 3 - 4, 3 != -1. The pair (1, 2) is a bad pair since 2 - 1 != 3 - 1, 1 != 2. The pair (2, 3) is a bad pair since 3 - 2 != 3 - 3, 1 != 0. There are a total of 5 bad pairs, so we return 5.   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> nums = [1,2,3,4,5] <br/> <strong>Output:</strong> 0 <br/> <strong>Explanation:</strong> There are no bad pairs.    <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= nums.length &lt;= 10<sup>5</sup></code></li>  <li><code>1 &lt;= nums[i] &lt;= 10<sup>9</sup></code></li> </ul>

# @param {Integer[]} nums
# @return {Integer}
def count_bad_pairs(nums)
  # all possible (i,j) are n*(n-1)/2; subtract good pairs from these
  # for a good pair, nums[i]-i == nums[j]-j, store nums[i]-i in the array
  n = nums.size
  num_idx_diff = Hash.new(0)
  nums.each_with_index { |num, i| num_idx_diff[num-i]+=1 }
  good_pairs = num_idx_diff.values.map {|x| x*(x-1)/2 }.sum
  n*(n-1)/2 - good_pairs
end
