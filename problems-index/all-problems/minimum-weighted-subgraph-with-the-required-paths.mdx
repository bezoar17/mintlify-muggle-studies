if paths are S1 and S2, they will coincide at a particular node
that node can be D, in which case, we have shortest path from D to S1 and S2
1-&gt;0-&gt;5 in Example 1
ans is S1-&gt;C + S2-&gt;C + C-&gt;D; where -&gt; is shortest path
we can do dijkstra 3 times here, from s1, s2, and from D
now for each node x, we have s1-&gt;x, s2-&gt;x and x-&gt;d
we can check for minimum amongst these for all nodes

```python3 title="Minimum Weighted Subgraph With the Required Paths" lines
class Solution:
    def minimumWeight(self, n: int, edges: List[List[int]], src1: int, src2: int, dest: int) -> int:
        adj = defaultdict(list)
        rev_adj = defaultdict(list)
        for u, v, w in edges:
            adj[u].append((v, w))
            rev_adj[v].append((u, w))
        
        def dijkstra(graph, source):
            pq = [(0, source)]
            dist = [float('inf')] * n
            
            while pq:
                d, node = heappop(pq)
                if d >= dist[node]:
                    continue
                dist[node] = d
                for neighbor, w in graph[node]:
                    if dist[neighbor] == float('inf'):
                        heappush(pq, (d + w, neighbor))
            return dist
        
        src1_dist = dijkstra(adj, src1)
        src2_dist = dijkstra(adj, src2)
        dist_from_d = dijkstra(rev_adj, dest)
        
        res = min((src1_dist[i] + src2_dist[i] + dist_from_d[i] for i in range(n)))
        return -1 if res == float('inf') else res
```
