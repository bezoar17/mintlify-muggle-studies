  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      treat split substrings as new chars, now both new strings should be anagrams.
subtrings can be directly treated as chars and just keep hash to check.
    </Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Rearrange K Substrings to Form Target String" lines
# @param {String} s
# @param {String} t
# @param {Integer} k
# @return {Boolean}
def is_possible_to_rearrange(s, t, k)
  chunk_size = s.size/k;
  sub_str_count = s.chars.each_slice(chunk_size).map(&:join).tally
  
  t.chars.each_slice(chunk_size).map(&:join).each do |sub_str|
    return false if sub_str_count[sub_str].to_i == 0
    sub_str_count[sub_str]-=1;
  end
  true
end
```
