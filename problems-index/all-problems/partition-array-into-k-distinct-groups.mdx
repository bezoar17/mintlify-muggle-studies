  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      Description is slightly off, each group should have exactly k elements, and all k should be distinct. So, we need n/k groups, and each should have k distinct elements. So, 2 simple checks, total size is a multiple of k. And freq of each number is at max n/k. 
    </Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Partition Array Into K-Distinct Groups" lines
# @param {Integer[]} nums
# @param {Integer} k
# @return {Boolean}
def partition_array(nums, k)
  nums.size % k == 0 && nums.tally.values.all? {|x| x<= nums.size/k}
end
```
