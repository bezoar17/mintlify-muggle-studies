      <p>Given an integer array <code>nums</code> and an integer <code>k</code>, return <em>the number of <strong>good</strong> subarrays of</em> <code>nums</code>.</p>  <p>A subarray <code>arr</code> is <strong>good</strong> if there are <strong>at least </strong><code>k</code> pairs of indices <code>(i, j)</code> such that <code>i &lt; j</code> and <code>arr[i] == arr[j]</code>.</p>  <p>A <strong>subarray</strong> is a contiguous <strong>non-empty</strong> sequence of elements within an array.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> nums = [1,1,1,1,1], k = 10 <br/> <strong>Output:</strong> 1 <br/> <strong>Explanation:</strong> The only good subarray is the array nums itself.   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> nums = [3,1,4,3,2,2,4], k = 2 <br/> <strong>Output:</strong> 4 <br/> <strong>Explanation:</strong> There are 4 different good subarrays: - [3,1,4,3,2,2] that has 2 pairs. - [3,1,4,3,2,2,4] that has 3 pairs. - [1,4,3,2,2,4] that has 2 pairs. - [4,3,2,2,4] that has 2 pairs.    <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= nums.length &lt;= 10<sup>5</sup></code></li>  <li><code>1 &lt;= nums[i], k &lt;= 10<sup>9</sup></code></li> </ul>

  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      Two pointer sliding window. When we extend r pointer, if nums[r] has appeared x times before, x pairs can be formed with this num value. We then extend l pointer to a point ensuring no. of pairs is still &gt;=k. When window is at l, any idx before this l value can be chosen with r pointer, so we add l+1 to the result. Then do this until we process all idxs as r.
    </Card>

    <CodeGroup dropdown>
```ruby title="Count the Number of Good Subarrays" lines
# @param {Integer[]} nums
# @param {Integer} k
# @return {Integer}
def count_good(nums, k)
  n = nums.size
  freq = Hash.new(0)
  pairs = 0 
  left = 0
  total_count = 0

  nums.each_with_index do |num, right|
    pairs+=freq[num]
    freq[num]+=1

    # shrink from left
    while pairs >= k
      total_count += n - right # All subarrays from left to right are valid
      freq[nums[left]] -=1
      pairs -= freq[nums[left]]
      left+=1
    end
  end
  
  total_count
end
```
