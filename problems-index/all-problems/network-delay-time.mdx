      <p>You are given a network of <code>n</code> nodes, labeled from <code>1</code> to <code>n</code>. You are also given <code>times</code>, a list of travel times as directed edges <code>times[i] = (u<sub>i</sub>, v<sub>i</sub>, w<sub>i</sub>)</code>, where <code>u<sub>i</sub></code> is the source node, <code>v<sub>i</sub></code> is the target node, and <code>w<sub>i</sub></code> is the time it takes for a signal to travel from source to target.</p>  <p>We will send a signal from a given node <code>k</code>.     <Card>
      Dijkstra's Implementation, without storing distances, just pushing popping in heap, last to pop will be max distance
</Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Network Delay Time" lines
# @param {Integer[][]} times
# @param {Integer} n
# @param {Integer} k
# @return {Integer}
def network_delay_time(times, n, k)
  edges = {}
  (1..n).each { |u| edges[u] = [] }
  times.each { |u, v, w| edges[u].append([v, w]) }

  heap = Containers::MinHeap.new([[0, k]])
  visited = Set.new
  total_weight = 0
  until heap.empty?
    weight_1, node_1 = heap.pop

    next if visited.include?(node_1)
    visited.add(node_1)

    total_weight = weight_1 # since its popping from min heap, last pop will be max
    edges[node_1].each do |node_2, weight_2|
      next if visited.include?(node_2)

      heap.push([weight_1 + weight_2, node_2])
    end
  end

  return -1 if visited.length != n

  total_weight 
end
```
