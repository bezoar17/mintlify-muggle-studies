      <p>You are given an array of strings <code>words</code> and a string <code>chars</code>.</p>  <p>A string is <strong>good</strong> if it can be formed by characters from <code>chars</code> (each character can only be used once for <strong>each</strong> word in <code>words</code>).</p>  <p>Return <em>the sum of lengths of all good strings in words</em>.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> words = [&quot;cat&quot;,&quot;bt&quot;,&quot;hat&quot;,&quot;tree&quot;], chars = &quot;atach&quot; <br/> <strong>Output:</strong> 6 <br/> <strong>Explanation:</strong> The strings that can be formed are &quot;cat&quot; and &quot;hat&quot; so the answer is 3 + 3 = 6.   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> words = [&quot;hello&quot;,&quot;world&quot;,&quot;leetcode&quot;], chars = &quot;welldonehoneyr&quot; <br/> <strong>Output:</strong> 10 <br/> <strong>Explanation:</strong> The strings that can be formed are &quot;hello&quot; and &quot;world&quot; so the answer is 5 + 5 = 10.    <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= words.length &lt;= 1000</code></li>  <li><code>1 &lt;= words[i].length, chars.length &lt;= 100</code></li>  <li><code>words[i]</code> and <code>chars</code> consist of lowercase English letters.</li> </ul>

```ruby title="Find Words That Can Be Formed by Characters" lines
# @param {String[]} words
# @param {String} chars
# @return {Integer}
def count_characters(words, chars)
  @char_h = chars.chars.tally
  
  def good?(word)
    temp_char_h = @char_h.dup
    word.chars.each do |ch|
      return false if temp_char_h[ch].to_i <= 0
      temp_char_h[ch] -= 1
    end
    true
  end

  words.select { |word| good?(word)}.map(&:size).sum
end
```
