      <p>You are given two strings <code>s1</code> and <code>s2</code> of equal length. A <strong>string swap</strong> is an operation where you choose two indices in a string (not necessarily different) and swap the characters at these indices.</p>  <p>Return <code>true</code> <em>if it is possible to make both strings equal by performing <strong>at most one string swap </strong>on <strong>exactly one</strong> of the strings. </em>Otherwise, return <code>false</code>.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> s1 = &quot;bank&quot;, s2 = &quot;kanb&quot; <br/> <strong>Output:</strong> true <br/> <strong>Explanation:</strong> For example, swap the first character with the last character of s2 to make &quot;bank&quot;.   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> s1 = &quot;attack&quot;, s2 = &quot;defend&quot; <br/> <strong>Output:</strong> false <br/> <strong>Explanation:</strong> It is impossible to make them equal with one string swap.   <p><strong class="example">Example 3:</strong></p>   <strong>Input:</strong> s1 = &quot;kelb&quot;, s2 = &quot;kelb&quot; <br/> <strong>Output:</strong> true <br/> <strong>Explanation:</strong> The two strings are already equal, so no string swap operation is required.    <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= s1.length, s2.length &lt;= 100</code></li>  <li><code>s1.length == s2.length</code></li>  <li><code>s1</code> and <code>s2</code> consist of only lowercase English letters.</li> </ul>

  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      If 2 strings differ at 0 or 2 places. And the differences in those places are just same chars but reversed.
    </Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Check if One String Swap Can Make Strings Equal" lines
# @param {String} s1
# @param {String} s2
# @return {Boolean}
def are_almost_equal(s1, s2)
  diffs = s1.chars.zip(s2.chars).select {|a,b| a!=b}  
  diffs.size == 0 || (diffs.size == 2 && diffs[0]==diffs[1].reverse )
end
```
