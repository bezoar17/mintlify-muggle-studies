      <p>Given two sorted arrays <code>nums1</code> and <code>nums2</code> of size <code>m</code> and <code>n</code> respectively, return <strong>the median</strong> of the two sorted arrays.</p>  <p>The overall run time complexity should be <code>O(log (m+n))</code>.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> nums1 = [1,3], nums2 = [2] <br/> <strong>Output:</strong> 2.00000 <br/> <strong>Explanation:</strong> merged array = [1,2,3] and median is 2.   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> nums1 = [1,2], nums2 = [3,4] <br/> <strong>Output:</strong> 2.50000 <br/> <strong>Explanation:</strong> merged array = [1,2,3,4] and median is (2 + 3) / 2 = 2.5.    <p><strong>Constraints:</strong></p>  <ul>  <li><code>nums1.length == m</code></li>  <li><code>nums2.length == n</code></li>  <li><code>0 &lt;= m &lt;= 1000</code></li>  <li><code>0 &lt;= n &lt;= 1000</code></li>  <li><code>1 &lt;= m + n &lt;= 2000</code></li>  <li><code>-10<sup>6</sup> &lt;= nums1[i], nums2[i] &lt;= 10<sup>6</sup></code></li> </ul>

  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      Easy to do once you implement https://leetcode.com/problems/find-median-from-data-stream/description/. 

Just use the solution above to do this, for all nums in the 2 arrays.
Idea is to use left max and right min heap, and always maintain the sizes of these heaps to be as close as possible.
</Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Median of Two Sorted Arrays" lines
# @param {Integer[]} nums1
# @param {Integer[]} nums2
# @return {Float}
def find_median_sorted_arrays(nums1, nums2)
  l_max_h = MaxHeap.new
  r_min_h = MinHeap.new

  (nums1 + nums2).each do |num|
    if r_min_h.size <= l_max_h.size
      l_max_h.push(num); r_min_h.push(l_max_h.pop)
    else
      r_min_h.push(num); l_max_h.push(r_min_h.pop)
    end
  end

  r_min_h.size > l_max_h.size ? r_min_h.min : (r_min_h.min + l_max_h.max)/2.0
end
```
