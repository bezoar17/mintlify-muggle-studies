    <Card>
      <p>You are given an integer array <code>nums</code>. The <strong>absolute sum</strong> of a subarray <code>[nums<sub>l</sub>, nums<sub>l+1</sub>, ..., nums<sub>r-1</sub>, nums<sub>r</sub>]</code> is <code>abs(nums<sub>l</sub> + nums<sub>l+1</sub> + ... + nums<sub>r-1</sub> + nums<sub>r</sub>)</code>.</p>  <p>Return <em>the <strong>maximum</strong> absolute sum of any <strong>(possibly empty)</strong> subarray of </em><code>nums</code>.</p>  <p>Note that <code>abs(x)</code> is defined as follows:</p>  <ul>  <li>If <code>x</code> is a negative integer, then <code>abs(x) = -x</code>.</li>  <li>If <code>x</code> is a non-negative integer, then <code>abs(x) = x</code>.</li> </ul>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> nums = [1,-3,2,3,-4] <br/> <strong>Output:</strong> 5 <br/> <strong>Explanation:</strong> The subarray [2,3] has absolute sum = abs(2+3) = abs(5) = 5.   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> nums = [2,-5,1,-4,3,-2] <br/> <strong>Output:</strong> 8 <br/> <strong>Explanation:</strong> The subarray [-5,1,-4] has absolute sum = abs(-5+1-4) = abs(-8) = 8.    <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= nums.length &lt;= 10<sup>5</sup></code></li>  <li><code>-10<sup>4</sup> &lt;= nums[i] &lt;= 10<sup>4</sup></code></li> </ul>

  <Tab title="Solution" icon="lightbulb" iconType="regular" >
    <Card>
      Kadane's algo, but also track negative max value. And keep taking max of both positive and negative extremes.
</Card>
  <Expandable title="code">
    <CodeGroup dropdown>
```ruby title="Maximum Absolute Sum of Any Subarray" lines
# @param {Integer[]} nums
# @return {Integer}
def max_absolute_sum(nums)
  max_abs_sum = 0
  pos_sum = 0
  neg_sum = 0

  nums.each do |num|
    pos_sum = [pos_sum+num, num].max
    neg_sum = [neg_sum+num, num].min
    max_abs_sum = [pos_sum, -neg_sum, max_abs_sum].max
  end

  max_abs_sum
end
```
