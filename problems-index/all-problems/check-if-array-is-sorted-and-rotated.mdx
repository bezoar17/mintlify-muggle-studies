      <p>Given an array <code>nums</code>, return <code>true</code><em> if the array was originally sorted in non-decreasing order, then rotated <strong>some</strong> number of positions (including zero)</em>. Otherwise, return <code>false</code>.</p>  <p>There may be <strong>duplicates</strong> in the original array.</p>  <p><strong>Note:</strong> An array <code>A</code> rotated by <code>x</code> positions results in an array <code>B</code> of the same length such that <code>B[i] == A[(i+x) % A.length]</code> for every valid index <code>i</code>.</p>   <p><strong class="example">Example 1:</strong></p>   <strong>Input:</strong> nums = [3,4,5,1,2] <br/> <strong>Output:</strong> true <br/> <strong>Explanation:</strong> [1,2,3,4,5] is the original sorted array. You can rotate the array by x = 2 positions to begin on the element of value 3: [3,4,5,1,2].   <p><strong class="example">Example 2:</strong></p>   <strong>Input:</strong> nums = [2,1,3,4] <br/> <strong>Output:</strong> false <br/> <strong>Explanation:</strong> There is no sorted array once rotated that can make nums.   <p><strong class="example">Example 3:</strong></p>   <strong>Input:</strong> nums = [1,2,3] <br/> <strong>Output:</strong> true <br/> <strong>Explanation:</strong> [1,2,3] is the original sorted array. You can rotate the array by x = 0 positions (i.e. no rotation) to make nums.    <p><strong>Constraints:</strong></p>  <ul>  <li><code>1 &lt;= nums.length &lt;= 100</code></li>  <li><code>1 &lt;= nums[i] &lt;= 100</code></li> </ul>

```ruby title="Check if Array Is Sorted and Rotated" lines
# @param {Integer[]} nums
# @return {Boolean}
def check(nums)
  n = nums.size
  double = nums+nums
  (0...n).any? { |i| double[i, n].sort == double[i, n] }
end
```
